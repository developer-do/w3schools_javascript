JSON Data Types

유효한 데이터 유형

JSON에서 값은 다음 데이터 유형중 하나여야 합니다.

  • a string
  • a number
  • object(JSON object)
  • array
  • boolean
  • null

JSON 값은 다음 데이터 유형 중 하나일 수 없습니다.

  • function
  • Date
  • undefined

JSON string

JSON의 문자열은 큰 따옴표로 작성해야 합니다.

    {"name":"John"}

JSON number

JSON의 숫자는 정수 또는 부동 소수점이어야합니다.

    {"age":30}

JSON object

JSON의 값은 객체가 될 수 있습니다.

    {
      "employee": {
        "name":"John",
        "age":30,
        "city":"New York"
      }
    }

JSON의 값으로 사용되는 객체는 JSON객체와 동일한 규칙을 따라야 합니다.

JSON array

JSON의 값은 배열 일 수 있습니다.

    {
      "employees": ["John", "Anna", "Peter"]
    }

JSON Boolean

JSON의 Boolean값은 true/false 일 수 있습니다.

    {"sale":true}

JSON null

JSON의 값은 null일 수 있습니다.

    {"middlename":null}
