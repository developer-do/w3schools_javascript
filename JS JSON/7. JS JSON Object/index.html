JSON Object

Object syntax

    { "name": "John", "age": 30, "car":null }

JSON 객체는 중괄호 {}로 묶습니다.

JSON 객체는 key / value 쌍으로 작성됩니다.

key는 문자열이여야 하며 value는 유효한 JSON 데이터 유형(string, number, object, array, boolean, null)이어야 합니다.

key와 value는 콜론으로 구분됩니다.

각 key / value 쌍은 쉼표로 구분됩니다.

객체 값에 접근하기

dot (.) 표기법을 사용하여 객체 값에 접근할 수 있습니다.

    myObj = { "name": "John", "age": 30, "car": null };
    x = myObj.name;

대괄호([]) 표기법을 사용하여 객체 값에 접근할 수 있습니다.

    myObj = { "name": "John", "age": 30, "car": null };
    x = myObj["car"];

객체 루팡하기

for-in 루프를 사용하여 객체 속성을 반복할 수 있습니다.

    myObj = {"name":"John", "age":30, "car":null};
    for(x in myObj) {
      document.getElementById("demo1").innerHTML += myObj;
    }

for0in 루프에서 대괄호 표기법을 사용하여 속성 값에 접근합니다.

    myObj = {"name":"john", "age":30, "car": null};
    for(x in myObj) {
      document.getElementById("demo2").innerHTML = myObj[x];
    }

중첩된 JSON 객체

JSON 객체의 값은 다른 JSON 객체가 될 수 있습니다.

    myObj = {
      "name":"john",
      "age":30,
      "cars": {
        "car1":"Ford",
        "car2":"BMW",
        "car3":"Fiat"
      }
    }

점 표기법이나 괄호 표기법을 사용하여 중첩 된 JSON 객체에 접근할 수 있습니다.

    x = myObj.cars.car2;
    // or:
    x = myObj.cars["car2"];

값 수정

점 표기법을 사용하여 JSON객체의 값을 수정할 수 있습니다.

    myObj1.cars.car2 = "Mercedes";

대괄호 표기법을 사용하여 JSON 객체의 값을 수정할 수도 있습니다.

    myObj1.cars["car2"] = "Mercedes";

객체 속성 삭제

delete 키워드를 사용하여 JSON객체에서 속성을 삭제하십시오.

    delete myObj.cars.car2;
