JSON Syntax

JSON 구문은 JavaScript 구문의 하위 집합입니다.

JSON 구문 규칙

JSON 구문은 JavaScript 객체 표기법 구문에서 파생됩니다.

  • 데이터는 name / value 가 쌍으로 이루어져 있습니다.
  • 데이터는 쉼표로 구분됩니다.
  • 중괄호는 객체를 유지합니다.
  • 대괄호는 배열을 유지합니다.

JSON 데이터 - 이름 및 값

JSON 데이터는 name / value 가 쌍으로 이루어져 작성됩니다.

name / value 쌍은 필드 이름(큰 따옴표로 묶임)과 콜론, 값 다음으로 구성됩니다.

    "name":"John"

JSON 이름에는 큰 따옴표가 필요합니다. 자바스크립트 이름은 그렇지 않습니다.

JSON - JavaScript 객체를 평가합니다.

JSON 형식은 JavaScript 객체와 거의 동일합니다.

JSON에서 key는 큰따옴표로 묶은 문자열이어야 합니다.

    JSON
    { "name":"John" }

JavaScript에서 key는 문자열, 숫자 또는 식별자 이름이 될 수 있습니다.

    JavaScript
    { name:"John"}

JSON value

JSON에서 값은 다음과 같은 데이터 유형중 하나여야 합니다.

  • a string
  • a number
  • an boejct(JSON obejct)
  • an array
  • a boolean
  • null

JavaScript 값에서 위의 모든 값과 다음을 포함한 기타 유효한 JavaScript 식이 될 수 있습니다.

  • a function
  • a Date
  • undefined

JSON에서 문자열 value은 큰 따옴표로 작성해야 합니다.

    JSON
    {"name":"John"}

JavaScript에서는 문자열 값을 큰 따옴표나 작은 따옴표로 작성할 수 있습니다.

    JavaScript
    {name:'John'}

JSON은 JavaScript 구문을 사용합니다.

JSON 구문은 JavaScript객체 표기법에서 파생되므로 JavaScript 내에서 JSON을 사용하려면 약간의 추가 소프트 웨어가 필요합니다.

JavaScript를 사용하면 다음과 같이 객체를 생성하고 객체에 데이터를 할당 할 수 있습니다.

    var person = { name : "John", age: 31, city: "New York" };

다음과 같이 JavaScript 객체에 접근할 수 있습니다.

    // returns John
    person.name;

다음과 같이 접근할 수도 있습니다.

    // returns John
    person["name"];

데이터는 다음과 같이 수정할 수 있습니다.

    person.name = "Gilbert";

다음과 같이 수정할수도 있습니다.

    person["name"] = "Gilbert";

이 자습서의 뒷부분에서 JavaScript 객체를 JSON으로 변환하는 방법을 배우게 됩니다.

JavaScript 배열을 JSON으로 사용

JavaScript 객체를 JSON으로 사용할 수 있는 것과 같은 방식으로 JavaScript 배열을 JSON으로 사용할 수도 있습니다.

이 튜토리얼의 뒷부분에서 JSON으로 배열에 대해 자세히 배울 것 입니다.

JSON 파일

  • JSON 파일의 파일 형식은 ".json"입니다.
  • JSON 텍스트의 MIME 유형은 "application / json" 입니다.
