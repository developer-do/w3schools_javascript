JSON - Introduction

JSON : JavaScript Object Notation.
JSON은 데이터를 저장하고 교환하기 위한 구문입니다.
JSON은 JavaScript 객체 표기법으로 작성된 텍스트 입니다.

데이터 교환

브라우저와 서버간에 데이터를 교환 할 때 데이터는 텍스트 일 뿐입니다.

JSON은 텍스트이며 모든 JavaScript 객체를 JSON으로 변환하고 JSON을 서버로 보낼 수 있습니다.

또한 서버에서 받은 JSON을 JavaScript 객체로 변환 할 수 있습니다.

이렇게 하면 복잡한 파싱 및 번역 작업 없이 데이터를 JavaScript 객체로 처리할 수 있습니다.

데이터 전송중

JavaScript 객체에 저장된 데이터가 있는 경우 객체를 JSON으로 변환하여 서버로 보낼 수 있습니다.

    var myObj = {name: "John", age: 31, city: "New York"};
    var myJSON = JSON.stringify(myObj);
    window.location = "demo_json.php?x=" + myJSON;

JSON.stringify() 함수에 대한 자세한 내용은 이 자습서 뒷부분에 있습니다.

데이터 수신

JSON 형식의 데이터를 받는 경우 JavaScript 객체로 변환 할 수 있습니다.

    var myJSON = '{"name":"John", "age":31, "city":"New York"}';
    var myObj = JSON.parse(myJSON);
    document.getElementById("demo").innerHTML = myObj.name;

데이터 저장

데이터를 저장할 때 데이터는 특정 형식이어야하며 저장할 위치와 관계없이 텍스트는 항상 합법적인 형식중 하나입니다.

JSON을 사용하면 JavaScript 객체를 텍스트로 저장할 수 있습니다.

    로컬 저장소에 데이터 저장
    // Storing data:
    myObj = {name: "John", age: 31, city: "New York"};
    myJSON = JSON.stringify(myObj);
    localStorage.setItem("testJSON", myJSON);

    // Retrieving data:
    text = localStorage.getItem("testJSON");
    obj = JSON.parse(text);
    document.getElementById("demo").innerHTML = obj.name;

JSON이란 무엇입니까 ?

  • JSON 약자는 JavaScript Object Notation이다.
  • JSON은 간단한 데이터 교환 형식입니다.
  • JSON은 "자체설명"하고 이해하기 쉽습니다.
  • JSON은 언어와 무관합니다.*

*
JSON은 JavaScript 구문을 사용하지만 JSON 형식은 텍스트만 사용합니다.
텍스트는 모든 프로그래밍 언어에서 데이터 형식으로 읽고 사용할 수 있습니다.

왜 JSON을 사용합니까?

JSON 형식은 텍스트 형식이므로 서버와 주고받을 수 있으며 모든 프로그래밍 언어에서 데이터 형식으로 사용됩니다.

JavaScript에는 JSON형식으로 작성된 문자열을 기본 JavaScript 객체로 변환하는 내장 함수가 있습니다.

JSON.parse()

따라서 JSON 형식의 서버에서 데이터를 받으면 다른 JavaScript 객체처럼 사용할 수 있습니다.
