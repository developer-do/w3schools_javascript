JSON.stringify()

JSON은 웹 서버와 데이터를 교환하는데 주로 사용됩니다.

데이터를 웹 서버로 보낼 때 데이터는 문자열이어야합니다.

JSON.stringify()로 JavaScript 객체를 문자열로 변환합니다.

JavaScript 객체 문자열로 변환하기

JavaScript에서 이 객체를 적용해보세요.

    var obj = { name: "John", age: 30, city: "New York" };

JSON.stringify() JavaScript 함수를 사용하여 문자열로 변환하십시오.

    var myJSON = JSON.stringify();

결과는 JSON 표기법을 따르는 문자열이 됩니다.

myJSON은 이제 문자열이며 서버에 보낼 준비가 되었습니다.

    var obj = { name: "John", age: 30, city: "New York" };
    var myJSON = JSON.stringify(obj);
    document.getElementById("demo").innerHTML = myJSON;

다음 장에서 JSON을 서버로 보내는 방법을 배웁니다.

JavaScript 배열 문자열화

JavaScript 배열을 문자열로 변환하는 것도 가능합니다.

자바스크립트에서 이 배열을 상상해 보십시오.

    var arr = [ "John", "Peter", "Sally", "Jane" ];

JSON.stringify() JavaScript 함수를 사용하여 문자열로 변환하십시오.

    var myJSON = JSON.stringify(arr);

결과는 JSON 표기법을 따르는 문자열 입니다.

myJSON은 이제 문자열이며 서버에 보낼 준비가 되었습니다.

    var arr = [ "John", "Peter", "Sally", "Jane" ];
    var myJSON = JSON.stringify(arr);
    document.getElementById("demo").innerHTML = myJSON;

예외

날짜 문자열 지정

JSON에서는 날짜 객체가 허용되지 않습니다. JSON.stringify() 함수는 모든 날짜를 문자열로 변환합니다.

    var obj = { name: "John", today: new Date(), city: "New York" };
    var myJSON = JSON.stringify(obj);

    document.getElementById("demo2").innerHTML = myJSON;

문자열을 수신기에서 날짜 객체로 다시 변환할 수 있습니다.

문자열 화 함수

JSON에서는 함수가 객체 값으로 허용되지 않습니다.

JSON.stringify() 함수는 키와 값 모두 JavaScript 객체에서 모든 함수를 제거시킵니다.

    var obj = { name: "John", age: function () {return 30;}, city: "New York"};
    var myJSON = JSON.stringify(obj);

    document.getElementById("demo3").innerHTML = myJSON;

JSON.stringify() 함수를 실행하기전에 함수를 문자열로 변환하면 생략할 수 있습니다.

    var obj = { name: "John", age: function () {return 30;}, city: "New York"};
    obj.age = obj.age.toString();
    var myJSON = JSON.stringify(obj);

    document.getElementById("demo4").innerHTML = myJSON;

JSON에서 함수를 사용하지 말아야합니다. 함수는 scope를 잃어버리고, 함수로 다시 변환하려면 eval()을 사용해야합니다.
