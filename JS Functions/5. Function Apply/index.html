JavaScript Function Apply

method 재사용

apply() method를 사용하면 다른 객체에서 사용할 수 있는 방법을 작성할 수 있습니다.

JavaScript apply() method

apply() method는 call() method와 유사하다

이 예에서는 fullName method를 person1에 적용합니다.

    var person = {
      fuilName: function(){
        return this.firstName + " " + this.lastName;
      }
    }
    var person1 = {
      firstName : "Mary",
      lastName : "Doe"
    }
    person.fullName.apply(person1); // Will return "Mary Doe";

call()과 apply의 차이점

차이점은 다음과 같습니다.

call() method는 별도로 인수를 가질수 있습니다.

apply() method는 인수를 배열로 사용합니다.

apply() method는 인수 목록대신 배열을 사용하려는 경우 매우 편리합니다.

인수를 사용한 apply() method

apply() method는 배열의 인수를 받는다.

  var person = {
    fullName : function(city, country) {
      return this.firstName + " " + this.lastName + "," + city + "," + country;
    }
  }
  var person1 = {
    firstName: "John",
    lastName: "Doe";
  }
  person.fullName.apply(person1, ["app", "ly"]);
  person.fullName.call(person1, "ca", "ll");

최대 배열 method 시뮬레이션

Math.max() method를 사용하여 (숫자 목록에서) 가장 큰 숫자를 찾을 수 있습니다.

  Math.max(1,2,3);

JavaScript 배열에는 max() method가 없으므로 Math.max() method를 적용할 수 있습니다.

  Math.max.apply(null, [1,2,3]);

첫 번쨰 인수(null)은 중요하지 않습니다. 이예제에서는 사용되지 않습니다.

다음 예제는 동일한 결과를 제공합니다.

  Math.max.apply(Math, [1,2,3]);  // Will also return 3
  ||
  Math.max.apply(" ", [1,2,3]);   // Will also return 3
  ||
  Math.max.apply(0, [1,2,3]);     // Will also return 3

JavaScirpt strict mode

JavaScript strict mede에서 apply() method의 첫 번째 인수가 객체가 아닌 경우에 이 객체는 호출 된 함수의 소유자(객체)가 됩니다. "비 엄격"모드 에서는 전역 객체가 됩니다.
