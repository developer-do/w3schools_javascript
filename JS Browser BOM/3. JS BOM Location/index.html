JavaScript Window Location

window.location 객체는 현재 페이지 주소(URL)를 가져오고 브라우저를 새 페이지로 리디렉션 하는 데 사용할 수 있습니다.

window location

window.location객체 에서는 윈도우 접두사 없이 작성할 수 있습니다.

  • window.location.herf는 현재 페이지의 href(URL)을 반환합니다.
  • window.location.hostname은 웹 호스트의 도메인 이름을 반환합니다.
  • window.location.pathname은 현재 페이지의 경로와 파일 이름을 반환합니다.
  • window.location.protocol은 사용 된 웹 프로토콜(http: || https:)을 반환합니다.
  • window.location.assign은 새 문서를 로드합니다.

window location href

window.location.href 속성은 현재 페이지의 URL을 반환합니다.

    현재 페이지의 href(URL)를 표시합니다
    document.getElementById("demo").innerHTML = "Page location is " + window.location.href;

window location hostname

window,location.hostname의 속성은 (현재 페이지) 인터넷 호스트의 이름을 반환합니다.

    document.getElementById("demo").innerHTML = "Page hostname is " + window.location.hostname;

window location pathname

window.location.pathname의 속성은 현재 페이지의 경로 이름을 반환합니다.

    현재 URL의 경로 이름 표시:
    document.getElementById("demo").innerHTML = "Page path is " + window.location.pathname;

window location protocol

window.location.protocol의 속성은 페이지의 웹 프로토콜을 반환합니다.

    document.getElementById("demo").innerHTML = "Page protocol is " + window.location.protocol;

window location port

window.location.port의 속성은 (현제 페이지) 인터넷 호스트 포트 수를 반환합니다.

    document.getElementById("demo").innerHTML = "Port number is " + window.location.port;

대부분의 브라우저는 기본 포트 번호를 표시하지 않습니다. (http: 80, https: 443)

window location assign() method는 새로운 문서를 로드합니다.

    function newDoc() {
      window.location.assign("https://www.w3schools.com");
    }
