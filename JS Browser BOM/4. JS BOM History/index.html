JavaScript Window History

window.history 객체는 브라우저 히스토리를 포함합니다.

window history

window.history 실행 객체는 윈도우 접두사 없이 작성할 수 있습니다.

사용자의 개인 정보를 보호하기 위해 JavaScript 가 이 객체에 접근하는 방법에는 제한이 있습니다.

몇가지 방법

  • history.back() - 브라우저에서 뒤로가기를 클릭하는 것과 같습니다.
  • history.forward() - 브라우저에서 앞으로가기를 클릭하는 것과 같습니다.

window history back

history.back() method는 history 목록의 이전 URL을 로드합니다.

이는 브라우저에서 뒤로 버튼을 클릭하는 것과 동일합니다.

    function goBack() {
      window.history.back()
    }

window history forward

history forward() method는 history 목록의 다음 URL을 로드합니다.

이는 브라우저에서 앞으로 버튼을 클릭하는 것과 같습니다.

    function goForward() {
      window.history.forward()
    }
