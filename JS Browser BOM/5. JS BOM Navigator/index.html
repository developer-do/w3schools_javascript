JavaScript Window Navigator

window.navigator 객체에는 방문자의 브라우저에 대한 정보가 들어 있습니다.

window navigator

window.navigator의 객체는 윈도우 접두사 없이 작성 할 수 있습니다.

몇가지 예

  • navigator.appName
  • navigator.appCodeName
  • navigator.platform

browser cookie

cookieEnabled 속성은, 그렇지 않은경우는 false, 쿠키를 사용하는 경우 true를 반환합니다.

    

    document.getElementById("demo1").innerHTML = "cookiesEnabled is " + navigator.cookieEnabled;

browser 응용프로그램 이름

appNAme의 속성은 브라우저 응용 프로그램 이름을 반환합니다.

    

    document.getElementById("demo2").innerHTML = "navigator.appName is " + navigator.appName;

이상하게도 "Netscape"는 IE11, Chrome, Firefox 및 Safari의 응용 프로그램 이름입니다.

browser 으용 프로그램 코드 이름

appCodeName의 속성은 브라우저의 응용 프로그램 코드의 이름을 반환합니다.

    

    document.getElementById("demo3").innerHTML = "navigator.appCodeName is " + navigator.appCodeName;

"Mozilla"는 Chrome, Firefox, IE, Safari 및 Opera의 애플리케이션코드 이름입니다.

browser engine

product속성은 브라우저 엔진의 제품 이름을 반환합니다.

    

    document.getElementById("demo4").innerHTML = "navigator.product is " + navigator.product;

이것에 의지하지 마십시오. 대부분의 브라우저는 제품이름으로 "Grcko"를 반환합니다.

browser version

appVersion의 속성은 브라우저에 대한버전 정보를 반환합니다.

    

    document.getElementById("demo5").innerHTML = navigator.appVersion;

browser agent

userAgent의 속성은 서버 브라우저가 보낸 사용자 agent header를 반환합니다.

    

    document.getElementById("demo6").innerHTML = navigator.userAgent;

경고!!!!

탐색기 객체의 정보는 종종 오도 될 수 있으므로 다음과 같은 이유로 브라우저 버전을 감지하느 데 사용하면 안됩니다.

  • 브라우저마다 동일한 이름을 사용할 수 있습니다.
  • 브라우저 소유자가 navigator 데이터는 변경할 수 있습니다.
  • 일부 브라우저는 사이트 테스트를 우회하는 것으로 잘못 판단합니다.
  • 브라우저는 브라우저보다 나중에 릴리스 된 새 운영체제를 보고 할 수 없습니다.

브라우저 플랫폼

platform 속성은 browser platform(운영체제)를 반환합니다.

    

    document.getElementById("demo7").innerHTML = navigator.platform;

browser language

language의 속성은 브라우저 언어를 반환합니다.

    

    document.getElementById("demo8").innerHTML = navigator.language;

browser가 online입니까?

online browser가 online인 경우 속성은 true를 반환합니다.

    

    document.getElementById("demo9").innerHTML = navigator.onLine;

Java를 사용할 수 있습니까?

Java를 사용하는 경우 javaEnabled() method는 true를 돌려줍니다.

    

    document.getElementById("demo10").innerHTML = navigator.javaEnabled();
