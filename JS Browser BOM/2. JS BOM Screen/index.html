JavaScript Window Screen

window.screen 객체는 사용자 화면에 대한 정보를 포함합니다.

Window Screen

window.screen의 객체는 윈도우 접두사 없이 작성할 수 있습니다.

등록정보

  • screen.width
  • screen.height
  • screen.availWidth
  • screen.availHeight
  • screen.colorDepth
  • screnn.pixelDepth

window screen width

screen.width 속성은 방문자의 화면 너비를 픽셀 단위로 반환합니다.

    화면의 너비를 픽셀 단위로 표시합니다.
    document.getElementById("demo").innerHTML = "Screen Width: " + screen.width + "px";

window screen height

screen.height 속성은 방문자의 화면 높이를 픽셀 단위로 반환합니다.

    화면의 높이를 픽셀 단위로 표시합니다.
    documenet.getElementById("demo1").innerHTML = "Screnn Height: " + screen.height + "px";

window screen 사용가능한 width

screen.availWdith 속성은 방문자의 화면 너비 (픽셀 단위)에서 Windows 작업표시줄과 같은 인터페이스 기능을 뺀 값을 반환합니다.

    화면의 사용가능한 너비를 픽셀 단위로 표시합니다.
    document.getElementById("demo2").innerHTML = "Available Screen Width : " + screen.availWidth + "px";

window screen 사용가능한 height

screen.availHeight 속성은 방문자의 화면 높이에서 windows작업 표시줄과 같은 인터페이스 기능을 땐 값을 반환합니다.

    document.getElementById("demo3").innerHTML = "Available Screen Height: " + screen.availHeight + "px";

window screen color 농도

screen.colorDepth 속성은 한 색상을 표시하는 데 사용되는 비트 수를 반환합니다.

현대의 모든 컴퓨터는컬러 해상도를 위해 24비트 또는 32비트 하드웨어를 사용합니다.

  • 24비트 = 16,777,216 다른 "트루 컬러"
  • 32qlxm = 4,294,967,296 다른 "딥 컬러"

이전 컴퓨터는 16비트를 사용 했습니다. : 66536가지의 "하이 컬러" 해상도

아주 오래된 컴퓨터와 오랜된 휴대폰은 8비트를 사용했습니다. : 256 가지 "VGA 색상"

    화면의 색상 수를 비트 단위로 표시합니다.
    document.getElementById("demo3").innerHTML = "Screen Color Depth: " + screen.colorDepth;

window screen pixel depth

screen.pixelDepth 속성은 화면의 픽셀 깊이를 반환합니다.

    화면의 픽셀 깊이를 비트 단위로 표시합니다.
    document.getElementById("demo5").innerHTML = "Screen Pixel Depth: " + screen.pixelDepth;

최신 컴퓨터의 경우 색상 수와 픽셀 수는 동일합니다.
