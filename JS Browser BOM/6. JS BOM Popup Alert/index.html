JavaScript Popup Boxes

자바스크립트에는 경고 상자, 확인상자 및 프롬프트 상자라는 세 종류의 팝업 상자가 있습니다.

alert box

사용자에게 정보가 전달되도록 하려면 경고 상자가 자주 사용됩니다.

경고 상자가 나타나면 사용자는 계속 진행하기 위해 "확인"을 클릭해야 합니다.

    통사론
    window.alert("sometext");

window.alert() method는 윈도우 접두사 없이 작성할 수 있습니다.

    alert("I am an alert box!");

Confirm Box

확인 상자는 사용자가 무언가를 확인하거나 수락하기를 원할 때 자주 사용됩니다.

확인 상자가 나타나면 사용자는 "확인"또는 "취소"를 클릭해야 계속할 수 있습니다.

사용자가 "확인"을 클릭하면 상자는 true를 반환합니다. 사용자가 "취소"를 클릭하면 상자는 false를 반환합니다.

    통사론
    window.confirm("sometext");

window.confirm() method는 윈도우 접두사 없이 작성 할 수 있습니다.

    if(confirm("Press a button!")) {
      txt = "You pressed OK!";
    } else {
      txt = "You pressed Cancel!";
    }

prompt Box

사용자가 페이지에 들어가기 전에 값을 입력하게 하려면 프롬프트 상자가 자주 사용됩니다.

프롬프트 상자가 나타나면 사용자는 입력 값을 입력 한 후 "확인" 또는 "취소"를 클릭해야 합니다.

사용자가 "확인"을 클릭하면 상자에 입력 된 값이 반환됩니다. 사용자가 "취소"를 클릭하면 상자는 null을 반환합니다.

    통사론
    window.prompt("sometext", "defaultText");

window.prompt() method는 윈도우 접두사 없이 작성 할 수 있습니다.

    var person = prompt("Please enter your name", "Harry Potter");

    if(person == null || person == "") {
      txt = "User cancelled the prompt.";
    } else {
      txt = "Hello " + person + "! How are you today?";
    }

줄 바꿈

팝업 상자 안에 줄 바꿈을 표시하려면 백슬래시 뒤에 문자 n을 사용하십시오.

    alert("Hello\nHow are you?");
