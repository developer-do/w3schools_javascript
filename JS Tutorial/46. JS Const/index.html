JavaScript Const

ECMAScript 2015

ES2015는 주요한 두 가지 새로운 JavaScript 키워드인 let과 const를 도입했습니다.

const로 정의된 변수는 재 할당 할 수 없다는 점을 제외하고 let 변수와 같이 동작합니다.

    const PI = 3.141592653589793;
    PI = 3.14;      // This will git an error
    PI = PI + 10;   // This whil alse give an error

block 범위

const 를 사용하여 변수를 선언한느 것은 block scope와 관련하여 let과 비슷합니다.

이 예에서 블록에서 선언된 x는 블록 외부에서 선언된 x와 다릅니다.

    var x = 10;
    // Here x is 10
    {
      const x = 2;
      // Here x is 2
    }
    // Here x is 10

이전 장에서 Block Scope에 대해 더 자세히 배울수 있습니다.

선언되었을 때 할당됨

JavaScript const 변수는 선언 될 때 값을 할당해야 합니다.

    잘못된
    const PI;
    PI = 3.14159265359;

    옳은
    const PI = 3.14159265359;

실제 상수가 아닙니다.

키워드 const는 약간 오해의 소지가 있습니다.

상수 값을 정의하지 않습니다. 값에 대한 상수 참조를 정의합니다.

이 때문에, 우리는 상수 프리미티브 값을 변경할 수는 없지만 상수 객체의 프로퍼티를 변경할 수 있습니다.

프리미티브 값

상수에 프리미티브 값을 할당하면 프리미티브 값을 변경할 수 없습니다.

    const PI = 3.141592653589793;
    PI = 3.14;      // This will give an error
    PI = PI + 10;   // This will also give an error

상수 객체 변경 가능

상수 객체의 속성을 변경할 수 있습니다.

    // You can create a const object:
    const car = {type:"Fiat", model:"500", color:"white"};

    // You can change a property:
    car.color = "red";

    // You can add a property:
    car.owner = "Johnson";

하지만 상수 객체는 재할당 할수는 없습니다.

    const car = {type:"Fiat", model:"500", color:"white"};
    car = {type:"Volvo", model:"EX60", color:"red"};    // ERROR

상수 배열 변경 가능

상수 배열의 요소를 변경할 수 있습니다.

    // You can create a constant array:
    const cars = ["Saab", "Volvo", "BMW"];

    // You can change an element:
    cars[0] = "Toyota";

    // You can add an element:
    cars.push("Audi");

하지만 상수를 재 할당 할 수는 없습니다.

    const cars = ["Saab", "Volvo", "BMW"];
    cars = ["Toyota", "Volvo", "Audi"];    // ERROR

브라우저 지원

IE 10 이하에서는 const 키워드가 지원되지 않습니다.

재 선언

JavaScript var 변수는 다시 선언한느 것은 프로그램의 어느 위치에서나 허용됩니다.

    var x = 2;    //  Allowed
    var x = 3;    //  Allowed
    x = 4;        //  Allowed

기본 var 또는 let변수를 const, 동일한 범위 또는 동일한 블록에 다시 선언하거나 재항당 하는 것은 허용되지 않습니다.

    var x = 2;         // Allowed
    const x = 2;       // Not allowed
    {
      let x = 2;     // Allowed
      const x = 2;   // Not allowed
    }

동일한 범위 또는 동일한 블록에서 기존의 const 변수를 다시 선언하거나 재 할당하는 것은 허용되지 않습니다.

    const x = 2;       // Allowed
    const x = 3;       // Not allowed
    x = 3;             // Not allowed
    var x = 3;         // Not allowed
    let x = 3;         // Not allowed
    
    {
      const x = 2;   // Allowed
      const x = 3;   // Not allowed
      x = 3;         // Not allowed
      var x = 3;     // Not allowed
      let x = 3;     // Not allowed
    }

const 다른 범위 또는 다른 블록에서 변수를 다시 선언하는 것은 허용됩니다.

    const x = 2;       // Allowed;

    {
      const x = 3;   // Allowed
    }
    
    {
      const x = 4;   // Allowed
    }

Hoisting

var로 정의된 변수는 맨 위에 올립니다.

var 변수를 선언하기 전에 사용할 수 있습니다.

    carName = "Volvo";    // You CAN use carName here
    var carName;

const로 정의된 변수는 맨 위에 올리지 않습니다.

Const가 선언되기 전에 변수를 사용 할 수 없습니다.

    carName = "Volvo";    // You can NOT use carName here
    const carName = "Volvo";
