JavaScript Array Methods

배열을 문자열로 변환

JavaScript method toString() 은 배열을 쉼표로 구분 된 배열 값의 문자열로 변환합니다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    document.getElementById("demo").innerHTML = fruits.toString();

join() method는 문자열에 모든 배열 요소를 결합한다.

join() toString()과 똑같이 동작하지만, 구분자를 지정할 수도 있습니다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    document.getElementById("demo2").innerHTML = fruits.join(" * ");

Popping and pushing

배열을 사용하여 작업하면 요소를 쉽게 제거하고 새 요소를 추가 할 수 있습니다.

이것은 pop과 push입니다.

배열에서 항목을 pop하거나 배열을 push합니다.

Popping

pop() method는 배열의 마지막 한 요소를 제거한다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    popping = fruits.pop();
    document.getElementById("demo3").innerHTML = popping;

pop() method는 "제거한 요소"값을 반환합니다.

    popping = fruits.pop();
    console.log(popping);

pushing

push() method는 배열(마지막)에 새 요소를 추가한다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    fruits.push("Kiwi");
    document.getElementById("demo4").innerHTML = fruits.join(" / ");

elements 이동

shift() method는 popping처럼 마지막이 아니라 첫번째 요소를 제거한다.

shift() method은 첫번째 배열 요소를 제거하고 다른 모든 배열 요소를 인덱스를 맞춰줌

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    var x = fruits.shift();
    document.getElementById("demo6").innerHTML = fruits.join(" / ");
    document.getElementById("demo7").innerHTML = x;

unshift() method는 배열(첫번째)에 새 요소를 추가하고 기존에 있는 요소의 index를 밀어준다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    var x = fruits.unshift("Lemon");
    document.getElementById("demo8").innerHTML = fruits.join(" / ");
    document.getElementById("demo9").innerHTML = x;

요소 변경하기

배열 요소는 해당 index 번호를 사용하여 접근합니다.

배열 index는 0부터 시작합니다. [0]은 첫번째 배열 요소이고 [1]은 두번째 요소이고 [2]는 세번째 요소입니다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    fruits[0] = "Kiwi";
    document.getElementById("demo10").innerHTML = fruits.join(" / ");

length 속성은 배열에 새 요소를 추가하는 쉬운방법을 제공합니다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    fruits[fruits.length] = "Kiwi";
    document.getElementById("demo11").innerHTML = fruits.join(" / ");

element remove

JavaScript 배열은 객체이기 때문에 JavaScript 연산자 delete를 사용하여 요소를 삭제 할 수 있습니다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    delete fruits[0];
    document.getElementById("demo12").innerHTML = fruits.toString();

delete를 사용하면 배열에 정의되지 않은 구멍이 생길 수 있습니다. delete대신에 pop() 또는 shift()를 사용하십시오.

Splicing an element

splice() method는 어레이에 새로운 항목을 추가 할 수 있다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    fruits.splice(2, 0, "Lemon", "Kiwi");
    document.getElementById("demo13").innerHTML = fruits.join(" , ");

첫번째 파라미터 (2)는 새요소를 추가 해야하는 위치를 정의한다.

두번째 파라미터 (0)은 제거되어야 하는 요소의 수를 정의한다.

나머지 매개변수 ("Lemon" , "Kiwi")는 추가할 새 요소를 정의한다.

splice() method는 삭제된 항목을 반환한다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    var x = fruits.splice(2, 0, "Lemon", "Kiwi");
    document.getElementById("demo14").innerHTML = x;

splice()를 사용하여 요소 제거

영리한 매개변수 설정을 사용하면 splice()를 사용하여 배열에 "구멍"을 남기지 않고 요소를 제거 할 수 있습니다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    fruits.splice(0, 1);
    document.getElementById("demo15").innerHTML = fruits.join(" , ");

첫 번째 매개변수 (0)은 새 요소를 추가 (삽입) 해야하는 위치를 정의합니다.

두 번째 매개변수 (1)은 제거되어야 하는요소 수를 정의합니다.

나머지 매개변수는 생략됩니다. 새로운 요소가 추가되지 않습니다.

배열 병함 (연결)

concat() method는 기존의 어레이들을 병합 (연접)하여 새로운 배열을 작성

    var myGirls = ["Cecilie", "Lone"];
    var myBoys = ["Emil", "Tobias", "Linus"];
    var myChildren = myGirls,concat(muBoys);

concat() method는 기존 배열을 변경하지 않습니다. 항상 새로운 배열을 반환합니다.

concat() method는 여러개의 배열 인수를 취할 수 있습니다.

    var arr1 = ["Cecilie", "Lone"];
    var arr2 = ["Emil", "Tobias", "Linus"];
    var arr3 = ["Robin", "Morgan"];
    var myChildren = arr1.concat(arr2, arr3);

concat() method는 값을 인수로 취할 수도 있습니다.

    var arr1 = ["Cecilie", "Lone"];
    var myChildren = arr1.concat(["Emil", "Tobias", "Linus"]);

배열 자르기

slice() method는 새로운 배열로 array의 일부를 잘라냅니다.

다음 예제는 배열요소 1의위치에 배열 일부를 잘라냅니다.

    var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
    var citrus = fruits.slice(1);

slice() method는 새로운 배열을 만듭니다. 소스 배열에서 요소를 저게하지 않습니다.

다음 예제는 배열요소 3의 위치에 시작하는 배열의 일부를 잘라냅니다.

    var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
    var citrus = fruits.slice(3);

slice() method는 slice(1,3)와 같은 두개의 인수를 사용할 수 있습니다.

그런 다음 이 method는 시작 인수에서 끝 인수까지 요소를 선택합니다.(끝 인수는 포함하지 않습니다.)

    var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
    var citrus = fruits.slice(1, 3);

첫 번째 예제에서와 같이 end 인수가 생략되면 slice() 메서드는 나머지 배열을 분할합니다.

    var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
    var citrus = fruits.slice(2);

자동 toString()

프리미티브 값이 예상되면 JavaScript는 자동으로 배열을 쉼표로 구분 된 문자열로 변환합니다.

배열을 출력하려고 할 때 항상 그렇게 반환됩니다.

이 두 예제는 동일한 결과를 생성합니다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    document.getElementById("demo").innerHTML = fruits.toString();
  
    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    document.getElementById("demo").innerHTML = fruits;

모든 JavaScript 객체에는 toString() 메서드가 있습니다.

배열에서 최대 값과 최소값 찾기

JavaScript 배열에서 가장 높은 값 또는 가장 낮은 값을 찾는 내장 할수는 없습니다.

가장 낮은 값은 [0] 가장 높은 배열값은 [arrayName.length - 1]이다.
