JavaScript String

JavaScript 문자열은 텍스트 저장 및 조작에 사용됩니다.

JavaScript 문자열

JavaScript 문자열은 따옴표 안에 0개이상의 문자로 작성됩니다.

    var x = "John Doe";

작은 따옴표나 큰 따옴표를 사용할 수 있습니다.

    var carName = "Volvo XC60";
    var carName = 'Volvo XC60';

문자열을 둘러싼 따옴표와 일치하지 않는 한 문자열 내에서 따옴표를 사용할 수 있습니다.

    var answer = "It's alright";
    var answer = "He is called 'Johnny'";
    var answer = 'He is called "Johnny"';

문자열 길이

문자열의 길이는 내장된 속성 length를 사용해서 찾을 수 있습니다.

    var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
    var sln = txt.length;

특수 문자

문자열은 따옴표로 작성해야 하므로 JavaScript는 다음 문자열을 오해합니다.

문자열은 "우리는 소위 입니다."라고 잘게 자릅니다.

이 문제를 피하느 해결책은 백 슬래시 이스케잎 문자를 사용하는 것 입니다.

백슬래시(\) 이스케이프 문자는 특수 문자를 문자열로 바꿔줍니다.

\' -> ' 가 된다.

\" -> " 가 된다.

\\ -> \ 가 된다.

문자열 \"은 문자열에 큰 따옴표를 삽입합니다.

    var x = "We are the so-called \"Vikings\" from the north";

문자열 \'은 문자열에 작은 따옴표를 삽입합니다.

    var x = 'It\'s alright';

문자열 \\ 는 문자열에 백슬래시를 삽입합니다.

    var x = "The character \\ is called backslash.";

6개의 다른 이스케이프 시퀸스가 JavaScript에서 유효합니다.

  • \b backspace
  • \f Form Feed
  • \n New Line
  • \r Carriage Return
  • \t Horizontal Tabulator
  • \v Vertical Tabulator

    document.getElementById("demo").innerHTML = "Hello " +
    "Dolly!";

문자열은 객체가 될 수 있습니다.

일반적으로 JavaScript 문자열은 리터럴에서 생성된 프리미티브 값입니다.

var firstName = "John";

그러나 문자열은 키워드 new를 사용하여 객체로 정의 할 수도 있습니다.

var firstName = new String("John");

    var x = "John";
    var y = new String("John");

    // typeof x will return string
    // typeof y will return object

문자열을 객체로 생성하지 마십시오. 실행속도가 느려집니다. new 키워드는 코드를 복잡하게 합니다. 이로인해 예상치 못한 결과가 발생할 수 있습니다.

== 연산자를 사용할 때 동등한 문자열은 같습니다.

    var x = "Ponny";
    var y = new String("Ponny");

    // (x == y) is true because x and y have equal values

=== 연산자를 사용하면 false입니다. === 연산자는 typeof와 value가 동일해야 합니다.

    var x = "Ponny";
    var y = new String("Ponny");

    // (x === y) is false because x and y have different types ( string and object )
  
    var x = new String("John");             
    var y = new String("John");

    // (x == y) is false because x and y are different objects

    var x = new String("John");             
    var y = new String("John");

    // (x === y) is false because x and y are different objects

(x == y)와 (x === y)의 차이점에 유의하십시오.

두 개의 JavaScript 객체를 비교하면 항상 false입니다.
