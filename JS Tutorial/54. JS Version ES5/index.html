이 장에서는 ES5의 가장 중요한 기능 중 일부를 소개합니다.

"use strict" 지침

"use strict" 는 JavaScript 코드가 "엄격 모드"로 실행 되어야 한다고 정의합니다.

엄격모드를 사용하면 예를들어 선언되지 않은 변수를 사용할 수 없습니다.

모든 프로그램에서 엄격모드를 사용할 수 있습니다. 선언되지 않은 변수를 사용하지 못하도록하는 등 더 깔끔한 코드를 작성하는 데 도움이 됩니다.

"use strict"는 단지 문자열 식입니다. 오래된 브라우저는 이해하지 못해도 오류를 던지지 않습니다.

String.trim()

String.trim()은 문자열 양쪽의 공백을 제거합니다.

    var str = "            Hello World!          ";
    console.log("str.trim() = ",str.trim());

Array.isArray()

객체가 배열인지 여부를 확인합니다.

    function myFunction(){
      var fruits = ["Banana", "Orange", "Apple", "Mango"];
      console.log(fruits);
    }

Array.forEach()

forEach() method는 각 배열 요소에 대해 함수를 한번 호출합니다.

    var txt = "";
    var numbers = [45, 4, 9, 16, 25];
    numbers.forEach(myFunction());

    function myFunction(val) {
      txt = txt + value + "\n";
    }

Array.map()

이 예제에서는 각 배열 값에 2를 곱합니다.

    var numbers1 = [45, 4, 9, 16, 25];
    var numbers2 = numbers1.map(function (value) {
      return value * 5;
    });

Array.filter()

이 예제에서는 값이 18보다 큰 요소로 새 배열을 만듭니다.

    var numbers = [45, 4, 9, 16, 25];
    var over18 = numbers.filter(function(a){
      console.log(a > 18);
    });

Array.reduce()

이 예제에서는 배열의 모든 숫자의 합계를 찾습니다.

    var numbers1 = [45, 4, 9, 16, 25];
    var sum = numbers1.redece(function(t,a){
      console.log(total + " + " + value + " =  "  + (total + value));
      return total + value;
    });

Array.reduceRight()

또한 이 예에서는 배열의 모든 숫자의 함계를 찾습니다.

    var number = [45, 4, 9, 16, 25];
    var sum = number.reduceRight(function(t, v){
      console.log(t + " + " + v + " = " + (t + v));
      return t + v;
    });

Array.every()

이 예에서는 모든 값이 18 이상인지 확인합니다.

    var number = [45, 4, 9, 16, 25];
    var allOver18 = number.every(function(a){
      return a > 18;
    });

Array.some()

이 예에서는 일부 값이 18 이상인지 확인합니다.

    var numbers = [45, 4, 9, 16, 25];
    var allOver18 = numbers.some(function(a){
      console.log(a > 18);
      return a > 18;
    });

Array.indexOf()

배열에서 요소 값을 검색하고 그 위치를 반환합니다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    var a = fruits.indexOf(/Apple/ig);

Array.lastIndexOf()

Array.lastIndexOf()는 Array.indexOf()와 같지만 배열의 끝에서 검색합니다.

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
    var a = fruits.lastIndexOf("Apple");

JSON.parse()

JSON의 일반적인 용도는 웹 서버에서 데이터를 받는 것 입니다.

웹 서버에서 이 텍스트 문자열을 받았다고 가정 해보십시오.

    '{"name":"John", "age":30, "city":"New York"}'

JSON.parse() JavaScript함수는 텍스트를 JavaScript 객체로 변환하는 데 사용됩니다.

    var obj = JSON.parse('{"name":"John", "age":30, "city":"New York"}');

JSON.stringify()

JSON의 일반적인 사용은 데이터를 웹 서버로 보내는 것입니다.

데이터를 웹 서버로 보낼 대 데이터는 문자열이어야 합니다.

자바스크립트에서 이 객체를 상상해 보십시오.

    var obj = {"name":"John", "age":30, "city":"New York"};

JSON.stringify() JavaScript 함수르 사용하여 문자열로 변환하십시오.

    var myJSON = JSON.stringify(obj);

Date.now()

Date.now()는 밀리 초 단위로 밀리 초를 반환합니다.

    var timInMSs = Date.now();

getters 와 setters 속성

ES5를 사용하면 속성 가져오기 또는 설정과 같은 구문을 사용하여 객체 메서드를 정의할 수 있습니다.

이 예제에서는 fullName이라는 속성에 대한 getter을 만듭니다.

    var person = {
      firstName: "John",
      lastName : "Doe",
      get fullName() {
        return this.firstName + " " + this.lastName;
      }
    };

다음 예제는 언어 속성에 대한 setter 및 getter를 만듭니다.

    var person = {
      firstName: "John",
      lastName : "Doe",
      language : "NO",
      get lang() {
        return this.language;
      },
      set lang(value) {
        this.language = value;
      }
    };

이 예제는 setter를 사용하여 대문자 언어 업데이트를 보호합니다.

    var person = {
      firstName: "John",
      lastName : "Doe",
      language : "NO",
      set lang(value) {
        this.language = value.toUpperCase();
      }
    };

    // Set an object property using a setter:
    person.lang = "en";

new object property method

object.defindProperty()는 ES5의 새로운 Object method입니다.

그것은 당신이 객체 속성을 정의하거나 속성의 값 및 / 또는 metaData를 변경 할 수 있게합니다.

    // Create an Object:
    var person = {
      firstName : "John",
      lastName  : "Doe",
      language  : "No"
    };

    // Change a Property:
    Object.defineProperty(person, "language", {
      value         : "EN",
      writable      : true,
      enumerable    : true,
      configurable  : true
    });

    // Enumberate Properties
    var txt = "";
    for(var x in person) {
      txt += person[x] + "\n";
    }

다음 예제는 enumeration 에서 language 속성을 숨기는 것을 제외하고는 동일한 코드입니다.

    // Create an Object:
    var person = {
      firstName : "John",
      lastName  : "Doe",
      language  : "No"
    };

    // Change a Property:
    Object.defineProperty(person, "language", {
      value         : "EN",
      writable      : true,
      enumerable    : false,
      configurable  : true
    });

    // Enumberate Properties
    var txt = "";
    for(var x in person) {
      txt += person[x] + "\n";
    }

이 예제는 setter와 getter를 생성하여 대문자 언어 없데이트를 보호합니다.

    /// Create an Object:
    var pers = {
      firstName: "John",
      lastName : "Doe",
      language : "NO"
    };

    // Change a Property:
    Object.defineProperty(pers, "language", {
      get : function() { return language },
      set : function(value) { language = value.toUpperCase()}
    });

    // Change Language
    pers.language = "en";

ECMAScript 5는 많은 new Object Methods를 JavaScript에 추가했습니다.

    ES5 new Object Methods
    // Adding or changing an object property
    Object.defineProperty(object, property, descriptor)

    // Adding or changing many object properties
    Object.defineProperties(object, descriptors)

    // Accessing Properties
    Object.getOwnPropertyDescriptor(object, property)

    // Returns all properties as an array
    Object.getOwnPropertyNames(object)

    // Returns enumerable properties as an array
    Object.keys(object)

    // Accessing the prototype
    Object.getPrototypeOf(object)

    // Prevents adding properties to an object
    Object.preventExtensions(object)
    // Returns true if properties can be added to an object
    Object.isExtensible(object)

    // Prevents changes of object properties (not values)
    Object.seal(object)
    // Returns true if object is sealed
    Object.isSealed(object)

    // Prevents any changes to an object
    Object.freeze(object)
    // Returns true if object is frozen
    Object.isFrozen(object)

문자열에 대한 속성 접근

charAt() method는 문자열의 지정된 인덱스(위치)에서 문자를 반환한다.

    var str = "HELLO WORLD";
    str.charAt(0);      // returns H

ECMAScript 5는 문자열에 대한 속성 접근을 허용합니다.

    var str = "HELLO WORLD";
    str[0];

문자열에 대한 속성 접근은 예측할 수 없을 수도 있습니다.
