JavaScript While Loop

지정된 조건이 참이면 루프는 코드 블록을 실행할 수 있습니다.

While Loop

while 루프는 지정된 조건이 true인 동안 코드 블록을 반복합니다.

    통사론

    while (condition) {
      // code block to be executed
    }

다음 예에서 루프의 코드는 변수 (i)가 10보다 작은 한 반복해서 실행됩니다.

    while(i < 10) {
      text += "The number is " + i;
      i++;
    }

조건에 사용 된 변수를 늘리는 것을 잊으면 루프가 끝나지 않습니다. 그러면 브라우저가 다운 됩니다.

Do / While Loop

do / while 루프는 while 루프의 변형입니다. 이 루프는 조건이 참인지 확인하기 전에 코드 블록을 한번 실행한 다음 조건이 참인 동안 루프를 반복합니다.

    통사론

    do {
      // code block to be executed
    }
    while (condition);

아래 예제는 do / while 루프를 사용합니다. code block은 조건이 테스트 되기 전에 실행되므로 조건이 false인 경우에도 루프는 항상 한 번 이상 실행됩니다.

    do {
      text += "The number is " + i;
      i++;
    }
    while(i < 10);

조건에 사용 된 변수를 늘리는 것을 잊지 마십시오. 그렇지 않으면 루프가 끝나지 않습니다.!

for 및 while 비교

이전 장의 for 루프를 읽은 경우 while 루프는 for 루프와 거의 동일하며 statement1과 statement3이 생략된 것을 알 수 있습니다.

이 예제의 루프는 for 루프를 사용하여 cars 배열에서 자동차 이름을 수집합니다.

    var cars = ["BMW", "Volvo", "Saab", "Ford"];
    var i = 0;
    var text = "";

    for(; cars[i]){
      text += cars[i] + "< br >";
      i++;
    }

이 예제의 루프는 while 루프를 사용하여 cars 배열에서 자동차 이름을 수집합니다.

    var cars = ["BMW", "Volvo", "Saab", "Ford"];
    var i = 0;
    var text = "";

    while (i < 10) {
      text += cars[i] + "< br >";
      i++;
    }
