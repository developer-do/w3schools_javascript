JavaScript 할당 연산자

할당연산자는 JavaScript 변수에 값을 할당합니다.

예)

할당 연산자는 변수에 값을 할당합니다.

    var x = 10;

+= 대입연산자는 변수에 값을 더하기 합니다.

    var x = 10;
    x += 5;

-= 대입연산자는 변수에 값을 빼기 합니다.

    var x = 10;
    x -= 5;

*= 대입연산자는 변수에 값을 곱하기 합니다.

    var x = 10;
    x *= 5;

/= 대입연산자는 변수에 값을 나누기 합니다.

    var x = 10;
    x /= 5;

%= 대입연산자는 변수에 값을 나누기한 값에 나머지를 구합니다.

    var x = 10;
    x %= 5;
