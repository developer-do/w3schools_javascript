JavaScript Introduction

JavaScript는 HTML 내용을 변경 할 수 있습니다.

많은 JavaScript HTML 메소드중 하나가 getElementById() 입니다.

이 예제는 이 메소드를 사용하여 HTML element (id = "demo")를 "검색" 하고 요소 내용 (innerHTML)을 "Hello JavaScript"로 변경합니다.

JavaScript can change HTML content.

JavaScript는 HTML 속성 값을 변경할 수 있습니다.

이 예제에서 JavaScript는 img 태그의 src 속성 값을 변경합니다.

JavaScript는 HTML Style을 변경 할 수 있습니다.

HTML 요소의 스타일을 변경하는 것은 HTML속성을 변경하는 변형입니다.

위의 버튼을 클릭해서 font-size를 변경해주세요.

JavaScript가 HTML 요소를 숨길 수 있습니다.

HTML 요소 숨기기는 display 스타일에서 변경하여 수행할 수 있습니다.
