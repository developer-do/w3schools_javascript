JavaScript Switch

switch문은 다른 조건에 따라 다르 ㄴ동작을 수행하는 데 사용됩니다.

JavaScirpt 전환 명령문

switch문을 사용하여 실핼할 많은 코드 블록중 하나를 선택하십시오.

    통사론

    switch(expression) {
      case x :
        // code block
        break;
      case y :
        // code block
        break;
      default :
        // code block
    }

이것이 작동하는 방법입니다.

  • 스위치 표현식은 한 번 평가 됩니다.
  • 표현식의 값은 각 경우의 값과 비교됩니다.
  • 일치하는 항목이 있으면 연관된 코드블록이 실행됩니다.
    getDay() method는 요일을 0에서 6사이의 숫자로 반환합니다.
    (일요일 = 0, 월요일 = 1, 화요일 = 2, ....)
    이 예에서는 요일 번호를 사용하여 요일 이름을 계산합니다.

    switch (new Date().getDay()) {
      case 0:
        day = "Sunday";
        break;
      case 1:
        day = "Monday";
        break;
      case 2:
        day = "Tuesday";
        break;
      case 3:
        day = "Wednesday";
        break;
      case 4:
        day = "Thursday";
        break;
      case 5:
        day = "Friday";
        break;
      case 6:
        day = "Saturday";
    }

the break keyword

JavaScript가 break 키워드에 도달하면 스위치 블록에서 빠져나옵니다.

이렇게하면 블록 내부에서 더 많은 코드 및 사례 테스트를 중단하게 됩니다.

견기가 발견되고 일이 끝나면 쉴 시간이다. 더이상의 시험은 필요없다.

break는 스위치 블록에 있는 나머지 모든 코드의 실행을 "무시"하기 때문에 많은 실행시간을 절약 할 수 있습니다.

스위치 블록에서 마지막 경우를 중단할 필요는 없습니다.블록은 어쨌든 거기서 끊어집니다.

the default keyword

default 키워드는 일치하지 않는 경우 실행하는 코드를 지정합니다.

    getDay() 메서드는 요일을 0에서 6사이의 숫자로 반환합니다.
    오늘 토요일 (6) 또는 일요일 (0)이 아닌 경우 기본 메세지를 작성하십시오.

    switch (new Date().getDay()) {
      case 6:
        text = "Today is Saturday";
        break; 
      case 0:
        text = "Today is Sunday";
        break; 
      default: 
        text = "Looking forward to the Weekend";
    }

default 는 스위치 블록에서 마지막 경우가 될 필요는 없습니다.

    switch (new Date().getDay()) {
      default: 
        text = "Looking forward to the Weekend";
        break;
      case 6:
        text = "Today is Saturday";
        break; 
      case 0:
        text = "Today is Sunday";
    }

default가 스위치 블록의 마지막 사례가 아닌경우, 기본 구분을 break하고 종료하십시오.

공통 코드 블록

다른 스위치 케이스에서 동일한 코드를 사용하기를 원할 때도 있습니다.

이 예제에서 4와 5는 동일한 코드 블록을 공유하고 0과 6은 다른 코드 블록을 공유합니다.

    switch(new Date().getDay()) {
      case 4:
      case 5: 
        text = "Soon it is Weekend";
        break;
      case 0:
      case 6:
        text = "It is Weekend";
        break;
      default:
        text = "Looking forward to the Weekend";
    }

세부 사항 전환

여려 사례가 사례값과 일치하면 첫 번째 사례가 선택된다.

일치하는 사례가 발견되지 않으면 프로그램은 기본 레이블로 계속 이동한다.

기본 라벨을 찾을수 없는경우 프로그램은 스위치 뒤에 있는 문장으로 계속 이동한다.

엄격한 비교

스위치 케이스는 strict 비교(===)를 사용합니다.

값은 일치해야하는 동일한 유형이어야합니다.

엄격한 비교는 피연산자가 동일한 유형인 경우에만 true가 될 수 있습니다.

이 예제에서는 x 와 일치하는 것이 없습니다.

    var x = "0";
    switch (x) {
      case 0:
        text = "Off";
        break;
      case 1:
        text = "On";
        break;
      default:
        text = "No value found";
    }
