JavaScript RegExp

정규 표현식은 검색 패턴을 형성하는 일련의 문자입니다.

텍스트 검색 및 텍스트 바꾸기 작업에 검색 패턴을 사용할 수 있습니다.

정규표현식이란 무었입니까?

정규표현식은 검색 패턴을 형성하는 일련의 문자입니다.

텍스트의 데이터를 검색 할 때 이 검색 패턴을 사용하여 검색 대상을 설명 할 수 있습니다.

정규 표현식은 단일 문자 또는 더 복잡한 패턴이 될 수 있습니다.

정규식으 사용하여 모든 유형의 텍스트 검색 및 텍스트 바꾸기 작업을 수행 할 수 있습니다.

    통사론

    /pattern/modifiers;
    
    var patt = /w3schools/i;

예제 설명:

/w3schools/i 는 정규식 입니다.

w3schools는 검색에 사용 될 패턴입니다.

i 는 수식어 입니다.(대 / 소문자를 구별하지 않도록 검색을 수정합니다.)

문자열 메소드 사용하기

JavaScript에서 정규 표현식은 search() 및 replace()의 두 string method와 함께 사용됩니다.

search() method는 표현식을 사용하여 일치 항목을 검색하고 일치 항목의 위치를 반환합니다.

replace() method는 패턴이 대체되는 수정 된 문자열을 리턴합니다.

문자열과 함께 문자열 search() 사용

searchi() method는 문자열에서 지정된 값을 검색하고 일치하는 위치를 반환합니다.

    문자열을 사용하여 문자열에서 "W3schools"를 검색합니다.

    var str = "Visit W3Schools!";
    var n = str.search("W3Schools");

문자열 search() 정규표현식 사용

    문자열에서 "w3schools"에 대해 대 / 소문자를 구분하지 않고 검색하려면 정규 표현식을 사용하십시오.
    
    var str = "Visit W3Schools";
    var n = str.search(/w3schools/i);

문자열 사용 replace()를 string으로 사용

replace() method는 지정된 값을 문자열의 다른 값으로 바꿉니다.

    var str = "Visit Microsoft!";
    var res = str.replace("Microsoft", "W3Schools");

String replace()를 정규식으로 사용

    대소 문자를 구분하지 않는 정규 표현식을 사용하여 Microsoft를 W3Schools 문자열로 대체하십시오.

    var str = "Visit Microsoft!";
    var res = str.replace(/microsoft/i, "W3Schools");

당신은 알아차리셨나요 ?

위의 메소드에서는 정규 표현식 인수(문자열 인수 대신)를 사용할 수 있습니다.
정규표현식은 검색을 훨씬 강력하게 만들 수 있습니다.(예: 대소문자를 굽누하지 않음)

정규식 표현식 수정자

수식어는 대문자와 소문자를 구별하는 글로벌 검색을 실시하기 위해서 사용할 수 있습니다.

Modifier Description
i 대/소문자 구분 안함 일치 수행
g 전역 일치 수행(첫 번째 일치 후 중지하지 않고 모든 일치 항목 찾기)
m 다중행(multi-line) 검색

정규식 pattern

[]대괄호는 일련의 문자를 찾는데 사용됩니다.

Expression Description
[abc] 괄호 사이에 문자 찾기
예를 들어 pattern [a-d]는 pattern[abcd]와 똑같이 동작하며, "brisket"의 'b'에 일치하고, "city"의 'c'에 일치합니다. pattern /[a-z.]+/와 /[\w]+/는 "test.i.ng" 전체 문자열이 일치합니다.
[0-9] 괄호 사이에 숫자 찾기
[0-9]는 [0123456789]와 동일하다.
(x|y) |로 분리된 대안 찾기
예를 들어, /green|red/는 "green apple"의 'green'에 대응되고, "red apple"의 'red'에 대응됩니다.

메타 문자는 특별한 의미의 문자입니다.

Metacharacter Description
\d 숫자 문자에 대응됩니다. [0-9]와 동일합니다.
예를 들어, /\d/ 또는 /[0-9]/sms "B2 is the suite number."에서 '2'에 대응됩니다.
\s 스페이스, 탭, 폼피드, 줄바꿈 문자등을 포함한 하나의 공백 문자에 대응됩니다.
예를 들어, /\s\w*/는 "foo bar"의 ' bar'에 대응됩니다.
\b 단어 시작 또는 끝에 대응됩니다.
예를 들어,
/\bm/은 "moon"의 'm'에 대응됩니다.
/oo\b/는 "moon"의 'oo'부분에 대응되지 않는데, 왜냐하면 'oo'를 뒤따라오는 'n'이 단어 문자이기 때문입니다.
/oon\b/는 "moon"의 'oon'에 대응됩니다. 왜냐하면, 'oon'은 문자열의 끝이라서, 뒤따라 오는 단어 문자가 없기 때문입니다.
/\w\b\w/는 어떤 것에도 일치하지 않습니다. 왜냐하면, 단어 문자는 절대로 비 단어 문자와 단어 문자 두개가 뒤 따라 올수 없기 때문입니다.
\uxxxx 16진수 xxxx로 지정된 유니코드 문자 찾기

수량 한정자는 수량을 정의합니다.

Quaniuifier Description
n+ 하나 이상의 n이 포함된 문자열 일치
예를 들어, /a+/ 는 "candy"의 'a'에 대응되고 "caaaaaaandy" 의 모든 'a'들에 대응되지만, "cndy" 내의 어느 부분과도 대응되지 않습니다.
n* 앞의 표현식이 0회 이상 연속으로 반복되는 부분과 대응됩니다. {0,}과 같은 의미입니다.
예를 들어, /bo*/는 "A ghost booooed"의 'boooo'와 대응되고, "A bird warbled"의 'b'에 대응되지만 "A goat grunted"내의 어느 부분과도 대응되지 않습니다.
n? 앞의 표현식이 0 또는 1회 등장하는 부분과 대응됩니다. {0,1}와 같은 의미입니다.
예를 들어, /e?le?/는 "angel"의 'el'에 대응되고, "angle"의 'le'에 대응되고 또한 "oslo"의 'l'에도 대응됩니다.

RegExp 객체 사용

JavaSCript에서 RegExp 객체는 미리 정의된 속성 및 메서드가 있는 정규 표현식 객체입니다.

test() method 사용하기

test() method는 RegExp표현식 method 입니다.

문자열에서 pattern을 검색하고 결과에 따라 true 또는 false를 반환합니다.

다음 예제에서는 문자열에서 "e"문자를 검색합니다.

    var patt = /e/;
    patt.test("The best things in life are free!");

    위의 2줄 코드와 동일합니다.
    /e/.test("The best things in life are free!")

exec() method 사용하기

exec() method는 RegExp 표현식 method 입니다.

문자열에서 지정된 패턴을 검색하고 발견된 텍스트를 객체로 반환합니다.

일치하는 항목이 없으면 빈(null) 객체를 반환합니다.

다음 예제에서는 문자열에서 "e"문자를 검색합니다.

    /e/.exec("The best things in life are free!");
