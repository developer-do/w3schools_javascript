JavaScript Where To

script 태그

HTML에서는 script와 /script 태그 사이에 JavaScript 코드를 삽입해야합니다.

이전 JavaScript 예제에서는 < /script type="text/javascript">유형 속성을 사용할 수 있습니다. type 속성은 필요하지 않습니다.JavaSCript는 HTML의 기본 스크립팅 언어입니다.

JavaScript 함수 및 이벤트

javaScript 함수는 JavaScript 코드 블록으로, "호출" 될때 실행 될 수 있습니다.

예를 들어, 사용자가 버튼을 클릭할 때와 같이 이벤트가 발생할 때 함수를 호출 할 수 있습니다.

head 또는 body의 자바스크립트

HTML 문서에 원하는수의 스크립트를 배치 할 수 있습니다.

스크립트는 HTML 페이지의 body 섹션 또는 head섹션에 또는 둘다 배치할 수 있습니다.

외부 JavaScript

스크립트를 외부 파일에 배치 할 수 있습니다.

    script src="main.js"

외부 javascript의 장점

스크립트를 외부 파일에 배치하면 몇가지 이점이 있습니다.

  • HTML과 코드를 구분합니다.
  • HTML과 JavaScript를 읽고 유지하기가 더 쉽습니다.
  • 캐시된 JavaScript로 인해 페이지 로드가 속도가 빨라질 수 있습니다.

한 페이지에 여러 스크립트 파일을 추가하려면 어러 스크립트 태그를 사용하시면 됩니다.

외부 스크립트 참조

외부 스크립트는 전체 URL 또는 현재 웹 페이지와 관련된 경로로 참조 할 수 있습니다.

이예에서는 전체 URL을 사용하여 스크립트에 연결합니다.

    script src="https://www.w3schools.com/js/yScript1.js"

이 예제는 현제 웹 사이트의 지정된폴더에 있는 스크립트를 사용합니다.

    script src="/js/myScript1.js"

이 예제는 현재 페이지와 같은 폴더에 있는 스크립트에 연결됩니다.

    script src="myScript1.js"
