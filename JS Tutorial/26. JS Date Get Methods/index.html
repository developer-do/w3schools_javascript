JavaScript Get Date methods

이러한 mehtod는 날짜 객체에서 정보를 가져오는데 사용할 수 있습니다.

Method Description
getFullYear() 연도를 4자리 숫자로 가져오기(yyyy)
getMonth() 월을 숫자로 가져오기(0-11)
getDate() 날짜를 숫자로 가져오기(1-31)
getHours() 시간 가져오기(0-23)
getMinutes() 시간 가져오기(0-59)
getSeconds() 초 가져오기(0-59)
getMilliseconds() 밀리초 가져오기(0-999)
getTime() 시간 확보(70년 1월 1일 이후 밀리초)
getDay() 요일을 숫자로 가져오기(0-6)
Date.now() 현재 시간

getTime() method

다음 getTime() method는 1970년 1월 1일 이후의 밀리초 수를 반환합니다.

    var d = new Date();
    document.getElementById("demo").innerHTML = d.getTime();

getFullYear() method

getFullYear() method는 네자리 숫자로 날짜의 연도를 반환합니다.

    var d = new Date();
    document.getElementById("demo1").innerHTML = d.getFullYear();

getMonth() method

getMonth() method는 숫자 0~11을 반환합니다.

    var d = new Date();
    document.getElementById("demo2").innerHTML = d.getMonth();

자바스크립트에서 첫번째달(1월)은 번호 0 이므로 마지막 달(12월)은 11을 반환합니다.

일련의 이름을 사용하고 getMonth()를 사용하여 월을 이름으로 반환하십시오.

    var d = new Date();
    var months = ["January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"];
    document.getElementById("demo3").innerHTML = months[d.getMonth()];

getDate() method

getDate() method는 수(1~31)와 같은 날짜의 일을 반환합니다.

    var d = new Date();
    document.getElementById("demo4").innerHTML = d.getDate();

gerHours() method

getHours() method는 수(0~23)와 같은 날짜의 시간을 반환합니다.

    var d = new Date();
    document.getElementById("demo5").innerHTML = d.getHours();

getMinutes() method

getMinutes() method는 수(0~59)와 같은 날짜의 분을 반환합니다.

    var d = new Date();
    document.getElementById("demo6").innerHTML = d.getMinutes();

getSeconds() method

getSeconds() method는 수(0~59)와 같은 날짜의 초를 반환합니다.

    var d = new Date();
    document.getElementById("demo7").innerHTML = d.getSeconds();

getMilliseconds() method

getMilliseconds() method는 날짜의 밀리초를 숫자 (0~999)로 반환합니다.

    var d = new Date();
    document.getElementById("demo8").innerHTML = d.getMilliseconds();

getDay() method

getDay() method는 숫자(0~6)와 같은 날짜의 요일을 반환합니다.

    var d = new Date();
    document.getElementById("demo9").innerHTML = d.getDay();

자바스크립트에서 세계의 일부 국가가 주의 첫날을 "월요일"로 간주하더라고 주중 첫 번쨰 날(0)은 "일요일"을 의미합니다.

이름 배열을 사용하여 getDay()를 사용하여 평일을 이름으로 반환 할 수 있습니다.

    var d = new Date();
    var days = ["Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday"];
    document.getElementById("demo10").innerHTML = days[d.getDay()];
