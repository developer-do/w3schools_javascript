JavaScript Date Formats

JavaScript 날짜 입력

일반적으로 다음 세 가지 유형의 JavaScript 날짜 입력 형식이 있습니다.

유형 예
ISO 날짜 "2015-03-25"(국제 표준)
짧은 날짜 '03/25/2015'
긴 날짜 '2015 년 3 월 25 일'또는 '2015 년 3 월 25 일'

ISO 형식은 JavaScript에서 엄격한 표준을 따릅니다.

다른 형식은 잘 정의되어 있지 않으며 브라우저 별로 다를 수 있습니다.

자바스크립트 날짜 출력

입력 형식과 관계 없이 JavaScript는 전체 텍스트 문자열 형식으로 날짜를 출력합니다.

JavaScript ISO 날짜

ISO 8601은 날짜와 시간 표현을 위한 국제 표준입니다.

ISO 8601 구문 (YYYY-MM-DD)도 기본 JavaScript 날짜 형식입니다.

    var d = new Date("2015-03-25");

계산 된 날짜는 귀하의 시간대를 기준으로 합니다.
시간대에 따라 위의 결과는 3월 24일과 3월 25일 사이에 달라집니다.

ISO 날짜 (년 and 월)

ISO 날짜는 요일(YYYY-MM)을 지정하지 않고 작성할 수 있습니다.

    var d = new Date("2015-03");

ISO 날짜 (만년)

ISO 날짜는 월 및 일(YYYY)업이 작성할 수 있습니다.

    var d = new Date("2015");

시간대

시간대를 지정하지 않고 날짜를 설정할 때 JavaScript는 브라우저의 시간대를 사용합니다.

날짜를 가져올 때 시간대를 지정하지 않으면 결과가 브라우저의 시간대로 변환됩니다.

다시 말해: GMT (그리니치 표준시)로 날짜 / 시간을 만들면 사용자가 미국 중서부를 탐색하면 날짜 / 시간이 CDT(중앙 미국 일광 절약 시간)로 변환됩니다.

JavaSCript 짧은 날짜

간단한 날짜는 다음과 같이 "MM/DD/YYYY"구문으로 작성됩니다.

    var d = new Date("03/25/2015");

경고!

    var d = new Date("2015-3-25");

    var d = new Date("2015/03/25");

    var d = new Date("25-03-2015");
    사용을 권하지 않습니다.

자바스크립트 긴 날짜

긴 날짜는 다음과 같이 "MMM DD YYYY"구문으로 작성되는 경우가 가장 많습니다.

    var d = new Date("Mar 25 2015");

날짜 입력 - 파싱 날짜

유효한 날짜 문자열이 있는 경우 Date.parse() method를 사용하여 밀리초로 변환 할 수 있습니다.

Date.parse()는 날짜와 1970 년 1월 1일 사이의 밀리초 수를 반환합니다.

    var msec = Date.parse("March 21, 2012");
    document.getElementById("demo").innerHTML = msec;

그런다음 밀리초 수를 사용하여 날짜 객체로 변환 할수 있습니다.

    var msec = Date.parse("March 21, 2012");
    var d = new Date(msec);
    document.getElementById("demo").innerHTML = d;
