JavaScript 유효성 검사 API

제약 조건 유효성 검사 DOM method

Property Description
checkValidity() 입력 요소에 유효한 데이터가 포함되어 있으면 true를 반환한다.
setCustomValidity() 입력 요소의 validationMessage 속성을 설정하십시오.

입력 필드에 유효하지 않은 데이터가 있으면 메시지를 표시하십시오.

    function myFunction() {
      var inpObj = document.getElementById("id1");
      if(!inpObj.checkValidity()) {
        document.getElementById("demo1").innerHTML = inpObj.validationMessage;
      }
    }

제약 조건 유효성 검사 DOM 속성

Property Description
validity 입력 요소의 유효성과 관련된 boolean 특성 포함
validationMessage 유효성이 거짓일 때 브라우저가 표시할 메시지 포함
willValidate 입력 요소의 유효성을 검사할지 여부를 표시한다.

유효성 속성

입력 요소의 validity 속성에는 데이터의 유효성과 관련된 여러 속성이 포함됩니다.

Property Description
customError 사용자 지정 유효성 메시지가 설정된 경우 true로 설정하십시오.
patternMismatch 요소와 값이 패턴 속성과 일치하지 않는 경우 true로 설정하십시오.
rangeOverflow 요소의 값이 최대 속성보다 크면 true로 설정하십시오.
rangeUnderflow 요소의 값이 최소 속성보다 작으면 true로 설정하십시오.
stepMismatch 요소의 단계 특성에 따라 요소 값이 유효하지 않은 경우 true로 설정하십시오.
tooLong 요소의 값이 maxLength 특성을 초과하는 경우 true로 설정하십시오.
typeMismatch 요소 유형 특성에 따라 요소 값이 유효하지 않은 경우 true로 설정하십시오.
valueMissing 요소(필수 특성을 가진)에 값이 없는 경우 true로 설정하십시오.
valid 요소 값이 유효한 경우 true로 설정하십시오.

예제들

입력 필드의 숫자가 100보다 큰 경우(입력의 최대 속성) 메시지를 표시하십시오.

    rangeOverflow 속성
    
    

    


    function myFunction2() {
      var txt = "";
      if (document.getElementById("id1").validity.rangeOverflow) {
        txt = "Value too large";
      }
      document.getElementById("demo").innerHTML = txt;
    }

입력 필드의 숫자가 100보다 작으면 (입력의 최소 속성) 메시지를 표시하십시오.

    
    

    


    function myFunction3() {
      var txt = "";
      if (document.getElementById("id1").validity.rangeUnderflow) {
        txt = "Value too small";
      }
      document.getElementById("demo").innerHTML = txt;
    }
