JavaScript Data Types

JavaScript 데이터 유형

JavaScript 변수는 숫자, 문자열, 객체 등 많은 데이터 유형을 보유 할 수 있습니다.

    var length = 16;                                // Number
    var lastName = "Johnson";                       // String
    var x = {firstName:"John", lastName: "Doe"};    // Object

데이터 유형 개념

프로그래밍에서 데이터 유형은 중요한 개념 입니다.

변수를 조작 할 수 있으려면유형에 대해 알고 있어야합니다.

데이터 유형이 없으면 컴퓨터가 안전하게 이 문제를 해결할 수 없습니다.

    var x = 16 + "Volvo";

16에 "Volvo"를 추가하는 것이 합리적입니까? 오류가 발생하거나 결과를 산출합니까?

JavaScript는 위의 예제를 다음과 같이 취급합니다.

    var x = "16" + "Volvo";

숫자와 문자열을 추가 할 때 JavaScript는 숫자를 문자열로 처리합니다.

JavaScript는 표현식을 왼쪽에서 오른쪽으로 평가합니다. 시퀸스가 다르면 다른 결과가 발생할 수 있습니다.

    var x = 16 + 4 + "Volvo";

    result = 20Volvo;

    var x = "Volvo" + 16 + 4;

    result = Volvo164;

첫 번째 예제에서 JavaScript는 16 및 4를 "Volvo"에 도달할 때 까지 숫자로 처리합니다.

두 번째 예제에서 JavaScript는 첫 번쨰 피연산자가 문자열 이므로 모든 피연산자는 문자열로 처리됩니다.

JavaScript 유형은 동적입니다.

JavaScript에는 동적 유형이 있습니다. 즉, 동일한 변수를 사용하여 다른 데이터 유형을 유지할 수 있습니다.

    var x;        // Now x is undefined;
    x = 5;        // Now x is a Number;
    x = "John";   // Now x is a String;

JavaScript 문자열

문자열 (또는 텍스트 문자열)은 "John Doe"와 같은 일련의 문자입니다.

문자열은 따옴표로 기록됩니다. 작은 따옴표나 큰 따옴표를 사용할 수 있습니다.

    var carName = "Volvo XC60";
    var carName = 'Volvo XC60';

문자열을 둘러싼 따옴표와 일치하지 않는 한 문자열 내에서 따옴표를 사용할 수 있습니다.

    var answer = "It's alright";
    var answer = "He is called 'Johnny'";
    var answer = 'He is called "Johnny"';

JavaScript number

JavaScript에는 한 가지 유형의 숫자만 있습니다.

숫자는 십진수 또는 십진법 없이 쓸수 있습니다.

    var x1 = 34.00;
    var x2 = 34;
    (x1 === x2) === true

매우 큰 또는 여분의 작은 숫자는 과학적 표기법으로 작성할 수 있습니다.

    var y = 123e5;      // 12300000
    var z = 123e-5;     // 0.00123

JavaScript boolean

boolean은 true || false의 두가지 값만 가질 수 있습니다.

    var x = 5;
    var y = 5;
    var z = 6;
    (x == y) // return true;
    (x == z) // return false;

boolean은 종종 조건부 테스트에 사용됩니다.

JavaScript Array

JavaScript 배열은 대괄호로 작성됩니다.

배열 항목은 쉼표로 구분됩니다.

다음 코드는 cars 라는 배열을 선언하고 3개의 항목 (자동차 이름)을 포함 합니다.1

    var cars = ["Saab", "Volvo", "BMW"];

배열 인덱스는 0부터 시작합니다. 즉, 첫번째 항목은 [0]이고 두번쨰 항목은 [1] 입니다.

JavaScript Object

JavaScript 객체는 중괄호로 작성됩니다.

객체 속성은 쉼표로 구분된 이름 : 값 으로 작성됩니다.

    var person = {firstName: "John", lastName: "Doe", age: 50, eyeColor: "blue"};

위 예제의 객체(사람)은 firstName, lastName, age, eyeColor의 4가지 속성을 가집니다.

연산자 유형

자바스크릡트 typeof 연산자를 사용하여 자바스크립트 변수의 유형을 찾을 수 있습니다.

typeof는 연산자 변수 또는 표현식ㅇ의 타입을 반환

    typeof ""         // Returns "string"
    typeof "John"     // Returns "string"

    typeof 0          // Returns "number"
    typeof 314        // Returns "number"

undefined

자바스크립트에서 값이 없는 변수의 값은 undefined입니다 typeof도 undefined입니다.

    var car;          // Value is undefined, type is undefined
    car = undefined;  // Value is undefined, type is undefined

"" 빈 값

빈 값은 undefined와는 아무런 관련이 없습니다.

빈 문자열에는 유효한 값과 유형이 모두 있습니다.

    var car = "";       // The value is "", the typeof is "string"

null

자바스크립트에서 null은 "아무것도"아닙니다. 그것은 "존재하지 않는 무언가" 이기로 되어 있습니다.

불행이도 JavaScript에서는 null의 데이터 유형이 객체입니다.

typeof null이 객체라는 점을 자바스클비트의 버그라고 생각할 수 있습니다. typeof null은 null이어야 합니다.

객체를 null로 설정하여 객체를 비울 수 있습니다.

    var person = {firstName:"John", lastName: "Doe", age: 50, eyeColor: "blue"};
    person = null;

    var person = {firstName:"John", lastName: "Doe", age: 50, eyeColor: "blue"};
    person = undefined;

undefined 와 null의 차이점

값이 동일하지만 유형이 다르다.

    typeof undefined    // undefined
    typeof null         // object

    null === undefined  // false
    null == undefined   // true

원시 데이터

원시데이터 값은 추가 속성 및 메서드가 없는 단일 단순 데이터 입니다.

연산자 유형은 다음 원시 유형중 하나를 반환 할 수 있습니다.

  • string
  • number
  • boolean
  • undefined
    typeof "John"              // Returns "string" 
    typeof 3.14                // Returns "number"
    typeof true                // Returns "boolean"
    typeof false               // Returns "boolean"
    typeof x                   // Returns "undefined" (if x has no value)

복잡한 데이터

연산자는 두가지 복잡한 유형중 하나를 반환할 수 있다.

  • function
  • object

typeof 연산자는 객체, 배열 및 null에 대해 object를 반환합니다.

typeof 연산자는 함수에 대해 객체를 반환하지 않습니다.

    typeof {name:'John', age:34} // Returns "object"
    typeof [1,2,3,4]             // Returns "object" (not "array", see note below)
    typeof null                  // Returns "object"
    typeof function myFunc(){}   // Returns "function"

typeof 연산자는 JavaScript 배열이 객체이기 때문에 배열에 대해 "Object"를 반환합니다.
