JavaScript String Nethods

String method는 String 작업을 도와줍니다.

String method 및 property

"John Doe"와 같은 원시 값은 객체가 아니기 때문에 property나 method를 가질 수 없습니다.

그러나 자바스크립트에서는 method와 property를 실행할 때 자바스크립트가 원시 값을 객체로 처리하기 때문에 method와 property를 원시 값에서도 사용할 수 있습니다.

String length

length property는 String의 length를 return 시킵니다.

    var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
    var sln = txt.length;

String에서 a string find하기

indexOf() method는 String에서 지정된 텍스트의 첫 번째 발생 index를 return합니다.

    var str = "Please locate where 'locate' occurs!";
    var pos = str.indexOf("locate");

JavaScript는 위치를 0에서 부터 계산합니다.
0은 문자열의 첫 번째 위치이고, 1은 두 번째, 2는 세 번째 입니다.

lastIndexOf() method는 마지막 문자열에서 지정된 텍스트의 위치의 index를 return 합니다.

    var str = "Please locate where 'locate' occurs!";
    var pos = str.lastIndexOf("locate");

indexOf() 및 lastIndexOf() method는 찾는 string가 없으면 -1을 return합니다.

    var str = "Please locate where 'locate' occurs!";
    var pos = str.lastIndexOf("John");

두가지 방법 모두 검색의 시작 위치로 두 번쨰 매개변수를 사용합니다.

    var str = "Please locate where 'locate' occurs!";
    var pos = str.indexOf("locate",15);

String 에서 String search

search() method는 지정된 값에 대한 string을 search하고 일치한 위치를 반환합니다.

    var str = "Please locate where 'locate' occurs!";
    var pos = str.search("locate");

알아 차리셨습니까 ?

indexOf()와 search() 두 method는 동일 합니까?

두개의 method는 동일한 인수(매개변수)를 받아들이고 동일한 값을 return합니다.

두 가지 방법이 동일 하지는 않습니다. 차이점은 다음과 같습니다.

  • search() method는 두번쨰 시작 위치 인수를 취할 수 없습니다.
  • indexOf() method는 강력한 검색값인 정규표현식을 사용할 수 없습니다.

String part return

문자열의 일부를 추출하는 방법에는 3가지가 있습니다.

  • slice (start, end)
  • substring (start, end)
  • substr (start, length)

slice() method

slice() method는 String의 일부를 추출하여 추출한 부분을 새 String으로 retunr합니다.

이 method는 시작 index(위치)와 마지막 index(위치)의 두 매개변수를 사용합니다.

이 예제는 문자열의 일부를 위치 7에서 위치 13으로 분할 합니다.

    var str = "Apple, Banana, Kiwi";
    var res = str.slice(7, 13);

매개변수가 음수이면 문자열 끝에서 부터 계산됩니다.

이 예제는 -12위치에서 -6위치 까지 문자열의 일부를 추출합니다.

    var str = "Apple, Banana, Kiwi";
    var res = str.slice(7, 13);

두번째 매개변수를 생략하면 method는 문자열의 나머지 부분을 잘라서 반환합니다.

    var res = str.slice(7);

또는 끝에서 부터 계산

    var res = str.slice(-12);

IE8 이하에서는 음수 위치가 작동하지 않습니다. 풕 IE

substring() method

substring()는 slice()와 비슷합니다.

차이점은 substring()이 음수 인덱스를 허용 하지 않는다는 것 입니다.

    var str = "Apple, Banana, Kiwi";
    var pos = str.substring(7, 13);

두 번째 매개변수를 생략하면 substring()는 문자열의 나머지 부분을 잘라냅니다.

substr() method

substr()은 slice()와 비슷합니다.

차이점은 두 번째 매개변수는 추출되는 부분의 length를 지정한다는 것입니다.

    var str = "Apple, Banana, Kiwi";
    var res = str.substr(7, 6);

두 번째 매개변수를 생략하면 substr()은 나머지 문자열을 잘라서 반환합니다.

    var str = "Apple, Banana, Kiwi";
    var res = str.substr(7);

첫번째 매개변수가 음수이면 위치는 문자열 끝에서 부터 계산됩니다.

    var str = "Apple, Banana, Kiwi";
    var res = str.substr(-4);

String 내용 바꾸기

replace() method는 문자열에서 다른 값으로 지정된 값을 대체합니다.

    var str = "Please visit Microsoft!";
    var n = str.replace("Microsoft", "W3Schools");

replace() method는 호출된 문자열을 변경하지 않습니다. 새 문자열을 반환합니다.

기본적으로 replace() method는 첫 번째 일치 항목만 바꿉니다.

    str = "Please visit Microsoft and Microsoft!";
    var n = str.replace("Microsoft", "W3Schools");

기본적으로 replace() method는 대 / 소문자를 구분합니다. MICROSOFT 는 작동하지 않습니다.

    str = "Please visit Microsoft!";
    var n = str.replace("MICROSOFT", "W3Schools");

대 / 소문자를 구분핮 ㅣ않으려면 /i 를 사용하는 정규표현식을 사용하십시오.

    str = "Please visit Microsoft!";
    var n = str.replace(/MICROSOFT/i, "W3Schools");

정규식을 따옴표 없이 작성됩니다.

모든 일치 항목을 바꾸려면 /g 플래그 (전역 일치)와 함께 정규식을 사용하십시오.

    str = "Please visit Microsoft!";
    var n = str.replace(/MICROSOFT/g, "W3Schools");

대 / 소문자로 변환

toUpperCase() method로 문자열을 대문자로 변환합니다.

    var text1 = "Hello World!";
    var text2 = text1.toUpperCase();

toLowerCase() method를 사용하면 문자열이 소문자로 변환됩니다.

    var text3 = text2.toLowerCase();

concat() method

concat() method는 두 개 이상의 문자열을 join합니다.

    var text1 = "Hello";
    var text2 = "World";
    var text3 = text1.concat(" ", text2);

concat() method 대신 덧셈 연산자가 사용될 수 있습니다. 이 두줄은 똑같습니다.

    var text = "Hello" + " " + "World!";
    var text = "Hello".concat(" ", "World!");

모든 string method는 새 string을 return합니다. 그들은 원래 string을 수정하지 않습니다.
공식적으로 : string는 변경 불가능합니다. string는 변경 될 수 없고 대체됩니다.

String trim()

String.trim() method는 문자열 양쪽끝의 공백을 제거합니다.

    var str = "      Hello World!       ";
    console.log(str.trim());

IE 8 이하에서는 String.trim()이 지원되지 않습니다.

IE 8 을 지원해야 하는 경우 trim()대신 String.replace를 정규식과 함께 사용할 수 있습니다.

    var str = "       Hello World!        ";
    alert(str.replace(/^[\s\uFEFF\xA0]+|[\s\uFEFF\xA0]+$/g, ''));

위의 바꾸기 솔루션을 사용하여 javaScriptStringportotype에 trim 함수를 추가할 수도 있습니다.

    if (!String.prototype.trim) {
      String.prototype.trim = function () {
      return this.replace(/^[\s\uFEFF\xA0]+|[\s\uFEFF\xA0]+$/g, '');
    };
    var str = "       Hello World!        ";
    alert(str.trim());

문자열에서 문자 추출하기

문자열에서 문자를 추출하는 방법에는 3가지가 있습니다.

  • charAt (position)
  • carCodeAt (position)
  • Property access []

charAt() method

charAt() method는 문자열에서 지정된 index(position)에 있는 문자를 돌려줍니다.

    var str = "HELLO WORLD";
    str.charAt(0);

charCodeAt() method

charCodeAt() method는 문자열에서 지정된 index에 있는 문자의 유니코드를 반환합니다.

이 method는 UTF-16 코드 (0에서 65535 사이의 정수)를 반환합니다.

    var str = "HELLO WORLD";  
    var ch = str.charCodeAt(0);

Property access []

ECMAScript 5 (2009)는 문자열에 대한 속성 엑세스 []을 허용합니다.

    var str = "HELLO WORLD";
    var ch = str[0];

속성 접근은 예측할 수 없을수도 있습니다.

  • IE7 이하에서는 작동하지 않습니다.
  • 문자열을 배열처럼 보이게 합니다 (그러나 그렇지 않습니다.)
  • 문자가 없으면 []는 undefined를 반환하고 charAt()은 빈 문자열을 반환합니다.
  • 읽기 전용합니다. str[0] = "A"는 오류를 주지 않습니다. (그러나 작동하지 않습니다.)
    var str = "HELLO WORLD";
    str[0] = "A";           // Gives no error, but does not work
    str[0];                 // returns H

문자열을 배열로 사용하려면 배열로 변환 할 수 있습니다.

문자열을 배열로 변환

split() method를 사용하면 문자열을 배열로 변환 할 수 있습니다.

    var txt = "a,b,c,d,e";
    txt.split(",");
    txt.split(" ");
    txt.split("|");

구분 기호가 생략되면 리턴된 배열은 전체 문자열을 index[0]에 포함시킵니다.

구분기호가 ""인 경우 반환되는 배열은 단일문자 배열입니다.

    var txt = "Hello";      // String
    var sp = txt.split(""); // Split in characters
