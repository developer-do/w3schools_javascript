JavaScript if else else if

조건문은 조건에 따라 다른 동작을 수행하는데 사용됩니다.

조건문

코드를 작성할 때 자주 다른 결정에 대해 다른 작업을 수행하려고 합니다.

코드에서 조건문을 사용하여 이를 수행 할 수 있습니다.

자바스크립트에는 다음과 같은 조건문이 있습니다.

  • if - 지정된 조건이 참인 경우 실행할 코드 블록을 지정
  • else - if의 조건이 거짓인 경우 실행할 코드블록을 지정하십시오.
  • else if - if의 조건이 거짓이고 다른조건으로 실행할 코드 블록을 지정
  • 사용 스위치를 코드의 많은 다른 블록을 지정하기 위해 실행합니다.

if 문

조건이 참인 경우 실행할 JavaScript 코드 블록을 지정하려면 if문을 사용하십시오.

    통사론

    if(condition) {
      // block of code to be executed if the condition is true
    }

소문자일 경우 주의 하십시오. 대문자 (If 또는 IF)는 JavaScript 오류를 생성합니다.

    시간이 18:00 미만이면 "좋은 하루" 인사말을 만드십시오.
    if(hour < 18) {
      greeting = "Good day";
    }

else 문

if 조건이 거짓인 경우 코드블록이 실행되도록 지정 블록에 다른 설명을 사용하십시오.

    통사론

    if(condition) {
      // block of code to be executed if the condition is true
    } else {
      // block of code to be executed if the condition is false
    }
  
    시간이 18 시간 미만이면 "좋은 하루" 인사말을 만들고, 그렇지 않은경우 "즐거운 저녁"인사말을 만듭니다.
    if(hour < 18) {
      greeting = "Good day";
    } else {
      greetion = "Good evening";
    }

else if 문

첫 번째 조건이 거짓인 경우 다른 조건을 사용하여 새 조건을 지정하십시오.

    통사론

    if(condition1) {
      // block of code to be executed if condition1 is true
    } else if(condition2) {
      // block of code to be executed if the confition1 is false and condition2 is true
    } else {
      // block of code to be executed if the condition1 is false and condition2 is false
    }
  
    시간이 10:00 미만인 경우 "Good morning"인사말을 작성하고 그렇지 않은 경우 20:00 미만인 경우 "Good day"인사말을 작성하고, 그렇지 않은 경우 "Good evening"을 작성하십시오.

    if(time < 10) {
      greeting = "Good morning";
    } else if (time < 20) {
      greeting = "Good day";
    } else {
      greetion = "Good evening";
    }
