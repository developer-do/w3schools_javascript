JavaScript Math Object

JavaScript Math 객체를 사용하면 숫자에 대한 수학적 작업을 수행 할 수 있습니다.

    Math.PI;      // returns 3.141592653589793

Math.round()

Math.round(x)는 가장 가까운 정수로 반올림 한 x의 값을 반환합니다.

    Math.round(4.7);    // returns 5
    Math.round(4.4);    // returns 4

Math.pow()

Math.pow(x,y)는 x의 값을 y의 거듭제곱으로 반환합니다.

    Math.pow(8, 2);     // returns 64

Math.sqrt()

Math.sqrt(x)는 x의 제곱근을 반환합니다.

    Math.sqrt(64);      // returns 8

Math.abs()

Math.abs(x)는 x의 절대 (양수)값을 반환합니다.

    Math.abs(-4.7);     // returns 4.7

Math.ceil()

Meth.ceil(x)는 x의 값을 가장 가까운 정수로 올림한 값을 반환됩니다.

    Math.ceil(4.4);     // returns 5

Math.floor()

Math.floor(x)는 x의 값을 가장 가까운 정수로 내림 한 값을 반환합니다.

    Math.floor(4.7);    // returns 4

Math.sin()

Math.sin(x)는 각도 x(사다리꼴로 주어진)의 sin(-1과 1사이의 값)을 반환합니다.

라디안 대신 도를 사용하려면 도를 라디안으로 변환해야합니다.

라디안 단위의 각도(라디안 단위) = 각도(도 * PI / 180)

    Math.sin(90 * Math.PI / 180);     // returns 1 (the sine of 90 degrees)

Math.cos()

Math.cos(x)는 각도 x의 코사인 (-1과 1사이의 값)을 반환합니다.(라디안 단위)

라디안 대신 도를 사용하려면 도를 라디안으로 변환해야 합니다.

라디안 단위의 각도(라디안 단위) = 각도(도 * PI / 180)

    Math.cos(0 * Math.PI / 180);     // returns 1 (the cos of 0 degrees)

Math.min() 및 Math.max()

Math.min() 및 Math.max()는 인수 목록에서 가장 낮은 값 또는 가장 높은 값을 찾는데 사용할 수 있습니다.

    Math.min(0, 150, 30, 20, -8, -200);   // returns -200
    Math.max(0, 150, 30, 20, -8, -200);   // returns 150

Math.random()

Math.random()은 0(포함)과 1(포함하지 않음) 사이의 난수를 반환합니다.

    Math.random();      // returns a random number

수학 속성 (상수)

JavaScript는 Math 객체로 액세스 할 수 있는 8개의 수학 상수를 제공합니다.

    Math.E        // returns Euler's number
    Math.PI       // returns PI
    Math.SQRT2    // returns the square root of 2
    Math.SQRT1_2  // returns the square root of 1/2
    Math.LN2      // returns the natural logarithm of 2
    Math.LN10     // returns the natural logarithm of 10
    Math.LOG2E    // returns base 2 logarithm of E
    Math.LOG10E   // returns base 10 logarithm of E

수학 생성자

다른 전역 개체와 달리 Math 개체에는 생성자가 없습니다. 메서드와 속성은 정적입니다.

Math 객체를 먼저 만들지 않고 모든 메서드와 속성(상수)을 사용할 수 있습니다.
