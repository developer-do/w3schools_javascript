자바스크립트 연산자

변수에 값을 할당하고 함께 추가하십시오.

    var x = 5;
    var y = 2;
    var z = x + y;

할당 연산자는 (=) 변수에 값을 할당한다.

    할당
    var x = 10;

가산 연산자 (+)는 숫자를 추가한다.

    var x = 5;
    var y = 2;
    var z = x + y;

곱셈 연산자 (*)는 숫자를 곱한다.

    var x = 5;
    var y = 2;
    var z = x * y;

JavaScript 산술 연산자

산술연산자는 숫자에 산술 연산을 수행하는 데 사용됩니다.

산술은 이미 알고있는 내용이니 눈으로 훑고 넘어가겠음

JavaScript 할당 연산자

할당 연산자는 JavaScript 변수에 값을 할당합니다.

Operator Example Same As
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y

또한 연산자(+ =)는 변수에 값을 추가한다.

    var x = 10;
    x += 5;
    x = 10 + 5;
    x = 15;

JavaScript 문자열 연산자

+ 연산자를 사용하여 문자열을 추가 (연결) 할 수도 있습니다.

    var txt1 = "John";
    var txt2 = "Doe";
    var txt3 = txt1 + " " + txt2;
    result = John Doe;

+ = 대입연산자를 사용하여 문자열을 추가 (연결) 할 수도 있습니다.

    var txt1 = "What a very";
    txt1 += "nice day";
    result = What a very nice day;

문자열에 사용되면 + 연산자는 연결 연산자라고 합니다.

문자열 몇 숫자 추가

두 개의 숫자를 추가하면 합계가 반환되지만 숫자와 문자열을 추가하면 문자열으로 반환됩니다.

    var x = 5 + 5;
    var y = "5" + 5;
    var z = "Hello" + 5;
    
    result x = 10;
    result y = 55;
    result z = Hello5;

숫자와 문자열을 더하면 문자열로 반환됩니다.
