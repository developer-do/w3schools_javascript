프로그래밍 언어에서 이러한 프로그래밍 명령을 명령문 이라고합니다.

자바스크립트 프로그램은 프로그래밍 state의 목록이다.

HTMl에서는 자바스크립트 프로그램이 웹 브라우저에 의해 실행됩니다.

JavaScript 문은 다음으로 구성됩니다.

값, 연산자, 표현식, 키워드 및 주석

이 문은 브라우저에 "Hello Dolly" 라고 쓰도록 지시합니다.

대부분의 JavaScript 프로그램에는 많은 JavaScript 문이 포함되어 있습니다.

Semicolons;

JavaScript 문은 세미콜론으로 구분됩니다.

각 실행 문 끝에 세미콜론을 추가 하십시오.

    var a, b, c;
    a = 5;
    b = 6;
    c = a + b;

세미콜론으로 구분하면 한 줄에 여러개의 명령문을 사용할 수 있습니다.

    a = 5; b = 6; c = a + b;

웹에서는 세미콜론이 없는 예제를 볼 수 있습니다.
세미콜론으로 끝나는 문장은 필수는 아니지만 적극 권장됩니다.

블록으로 그룹화된 구문을 찾을 수 있는 곳은 JavaScript 함수입니다.
