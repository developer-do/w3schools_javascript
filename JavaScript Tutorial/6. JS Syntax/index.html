JavaScript 구문

자바스크립트 구문은 자바스크립트 프로그램이 어떻게 구성되는지에 대한 일련의 규칙입니다.

    var x, y;
    x = 5; y =6;
    z = x + y;

자바스크립트 값

JavaScript 구문은 고정 값과 가변 값이라는 두가지 유형의 값을 정의합니다.

고정 값을 리터럴 이라고 하고, 변수 값을 변수라고 합니다.

자바스크립트 리터럴

고정 값 작성의 가장 중요한 규칙은 다음과 같습니다.

숫자는 십진법의 유무와 관계없이 쓰입니다.

    10.50

    1001

문자열은 이중 따옴표 또는 작음 따옴표로 작성된 텍스트 입니다.

    "John Doe"

    'John Doe'

자바스크립트 변수

프로그래밍 언어에서 변수는 데이터 값을 저장하는데 사용됩니다.

JavaScript는 var 키워드를 사용하여 변수를 선언합니다.

= 기호를 사용하여 변수에 값을 할당합니다.

이 예에서 x는 변수로 저의됩니다. 그런다음 x에 값 6이 지정됩니다.

    var x;
    
    x = 6;

자바스크립트 연산자

JavaScript는 산술연산자 (+ - * /)를 사용하여 값을 계산합니다.

    (5 + 6) * 10

JavaScript는 할당 연산자 (=)를 사용하여 변수에 값을 할당합니다.

    var x, y;
    x = 5;
    y = 6;

JavaScript 표현식

표현식은 값으로 계산되는 값, 변수 및 연산자의 조합입니다.

계산을 평가라고합니다.

예를들어, 5 * 10은 50으로 평가 됩니다.

    5 * 10

표현식에는 변수 값이 포함될 수도있습니다.

    var x = 5;
    x * 10;

값은 숫자 및 문자열과 같은 다양한 유형이 될 수 있습니다.

예를 들어, "John" + " " + "Doe"는 "John Doe"로 평가 됩니다.

    "John" + " " + "Doe";

자바스크립트 키워드

JavaScript 키워드는 수행 작업을 식별하는데 사용됩니다.

var에 키워드는 브라우저에서 변수를 생성하도록 지시합니다.

    var x,y;
    x = 5 + 6;
    y = x * 10;

자바스크립트 주석

모든 JavaScript문이 "실행" 되는건 아닙니다.

이중 슬래쉬 "//" || /* */ 사이의 코드는 주석으로 처리됩니다.

주석은 무시되고 실행되지 않습니다.

    var x = 5; // I will be executed
    // var x = 6; I will NOT be executed

자바스크립트 식별자

식별자는 이름입니다.

자바스크립트에서 식별자는 변수(및 키워드, 함수, 라벨)의 이름을 짓는데 사용된다.

대부분의 프로그래밍 언어에서는 이름에 대한 규칙이 거의 동일합니다.

자바스크립트에서 첫 번째 문자는 (문자) 또는 (_) 또는 ($) 달러 기호여야 합니다.

후속 문자는 문자, 숫자, 밑줄 또는 달러기호일 수 있습니다.

숫자는 첫 번째 문자로 허용되지 않습니다. 이 방법으로 자바스크립트는 식별자와 숫자를 쉽게 구별할 수 있습니다.

자바스크립트는 대소문자를 구분합니다.

모든 자바스크립트 식별자는 대소 문자를 구분합니다.

변수 lastName과 lastname은 두개의 다른 변수입니다.

    var lastname, lastName;
    lastName = "Doe";
    lastname = "Peterson";

VAR || Var는 키워드 var로 해석하지 않습니다.

자바스크립트 camel표시법

역사적으로 프로그래머는 여러 단어를 하나의 변수 이름으로 결합하는 다양한방법을 사용했습니다.

하이폰

first-name, last-name, master-card, inter-city

자바스크립트에서는 하이폰을 사용할 수 없습니다. 뺄셈을 위해 예약되어있습니다.

underscore

first_name, last_name, master_card, inter_city

Upper Camel Case(Pascal Case)

FirstName, LastName, MasterCard, InterCity

Lower Camel Case

firstName, lastName, masterCard, interCity

자바스크립트 문자 세트

자바스크립트는 유니코드 문자 집합을 사용합니다.

유니코드는 세계의 모든 문자, 구두점 및 기호를 (거의) 포함합니다.
