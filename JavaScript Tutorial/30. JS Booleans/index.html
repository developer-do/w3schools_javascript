JavaScript Booleans

JavaScript Boolean은 true 또는 false의 두 값중 하나를 나타냅니다.

Boolean value

매우 자주 프로그래밍에서 두 가지 값중 하나만 가질 수 있는 데이터 유형이 필요합니다.

  • 예 / 아니오
  • ON / OFF
  • 허위 사실

이를 위해 JavaScript에는 Boolean 데이터 유형이 있습니다. true 또는 false값만 사용할 수 있습니다.

Booleand() 함수

Boolean() 함수를 사용하여 표현식(또는 변수)이 참인지 확인할 수 있습니다.

    Boolean(10 > 9)   // returns true

    (10 > 9)          // also returns true
    10 > 9            // also returns true

비교 및 조건

JS 비교 장에서는 비교 연산자에 대한 전체 개요를 제공합니다.

JS 조건 장에서는 조건문에 대한 전체 개요를 제공합니다.

"value"가 있는 것은 모두 참입니다.

    100

    3.14

    -15

    "Hello"

    "false"

    7 + 1 + 3.14

"value"가 없는 것은 모두 거짓입니다.

    boolean 값이 0이면 false 입니다.
    var x = 0;
    Boolean(x);     // returns false;

    boolean값이 -0이면 false 입니다.
    var x = -0;
    Boolean(x);     // returns false;

    boolean값이 ""(빈 문자열)이면 false 입니다.
    var x = "";
    Boolean(x);     // returns false;
    
    boolean값이 undefined이면 false 입니다.
    var x = undefined;
    Boolean(x);     // returns false;

    boolean값이 null이면 false 입니다.
    var x = null;
    Boolean(x);     // returna false;

    boolean값이 false이면 false 입니다.
    var x = false;
    Boolean(x);     // returns false;

    boolean값이 NaN이면 false 입니다.
    var x = 10 / "H";
    Boolean(x);     // returns false;

객체가 될 수 있는 Boolean

일반적으로 자바스크립트 Boolean은 리터럴에서 생성된 프리미티브 값입니다.

var x = false;

그러나 booleans는 키워드 new를 사용하여 객체로 정의 할 수도 있습니다.

var y = new Boolean(false);

    var x = false;
    var y = new Boolean(false);

    // typeof x returns boolean
    // typeof y returns object

Boolean 객체를 만들지 마십시오. 실행 속도가 느려집니다. new 키워드는 코드를 복잡하게 합니다. 이로인해 예상치 못한 결과가 발생할 수 있습니다.

== 연산자를 사용하면 동등한 부울 값은 같습니다.

    var x = false;
    var y = new Boolean(false);

    // (x == y) is true because x and u have equal values

=== 연산자를 사용할 때 == 연산자는 형식과 값에서 동등성을 기대하므로 동등한 불린은 동일하지 않습니다.

    var x = false;
    var y = new Boolean(false);

    // (x === y) is false because x and y have different types

객체를 비교할 수 없습니다.

    var x = new Boolean(false);
    var y = new Boolean(false);

    // (x == y) is false because objects cannot be compared

두개의 JavaScript 객체를 비교하면 항상 false가 반환됩니다. 객체는 비교 자체를 할 수 없습니다.
