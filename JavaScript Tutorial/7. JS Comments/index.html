JavaScript Comments

JavaScript 주석을 사용하여 JavaScript 코드를 설명하고 더 쉽게 읽을 수 있습니다.

대안 코드를 테스트 할 때 JavaScript 주석을 사용하여 실행을 방지 할 수도 있습니다.

한 줄 주석

한 줄 주석은 //로 시작합니다.

//와 줄 끝 사이의 모든 텍스트는 JavaScript에 의해 무시됩니다. (실행되지 않음)

이 예제에서는 각 코드 줄 앞에 한 줄 주석을 사용합니다.

    // Change heading :
    document.getElementById("myH").innerHTML = "My First Page";
    // Change paragraph :
    document.getElementById("myP").innerHTML = "My first paragraph";

이 예제에서는 코드를 설명하기 위해 각 행의 끝에 단일 행 주석을 사용합니다.

    var x = 5;     // Declare x, give it the value of 5
    var y = x + 2; // Declaer y, give it the value of x + 2

여러줄 주석

여러 줄 주석은 "/*" 로 시작하고 "*/" 로 끝납니다.

/*와 */의 사이에 모든 텍스트는 JavaScript에 의해 무시됩니다.

이 예제는 여러줄의 주석 (주석 블록)을 사용하여 코드를 설명합니다.

    /*
      The code below will change
      the heading with id = "myH"
      and the paragraph with id = "myP"
      in my web page:
    */
    document.getELementById("myH").innerHTML = "My First Page";
    document.getELementById("myP").innerHTML = "My first paragraph";

한 줄 주석을 사용하는 것이 가장 일반적입니다. 블록주석은 공식 문서화에 자주 사용됩니다.

실행 방지를 위한 주석사용

코드 실행을 방지하기 위해 주석을 사용하면 코드 테스팅에 적합합니다.

코드 행 앞에 //를 추가하면 코드 행이 실행 가능 행에서 주석으로 변경됩니다.

이 예제에서는 코드 줄 중 하나의 실행을 방지하기 위해 //를 사용합니다.

    // document.getElementById("myH").innerHTML = "My First Page";
    document.getElementById("myP").innerHTML = "My first paragraph";

이 예제는 주석 블록을 사용하여 여러줄의 실행을 방지합니다.

    /* 
      document.getElementById("myH").innerHTML = "My First Page";
      document.getElementById("myP").innerHTML = "My first paragraph";
    */
