JavaScript Break and Continue

break 문은 루프에서 "나옵니다."

continue 문은 루프에서 하나의 반복을 "건너 뜁니다."

Break 문

이 자습서의 이전 장에서 사용 된 break 문은 이미 보았습니다. switch() 문에서 "나오기" 위해 사용되었습니다.

break 문은 루프 밖으로 점프하는 데 사용될 수도 있습니다.

break 문은 루프를 중단하고 루프(있는 경우)후 계속 실행합니다.

    for(var i = 0; i < 10; i++) {
      if( i === 3 ) {
        break;
      }
      text += "The number is " + i + "< br/ >";
    }

continue 문

continue은 특정 조건이 발생하는 경우 하나의 반복(루프에서)을 깨고 다음 반복을 반복합니다.

이 예제는 값 3을 건너 뜁니다

    for(var i = 0; i < 10; i++) {
      if(i === 3) {
        continue;
      }
      text += "the number is " + i + "< br >";
    }

JavaScript label

JavaScript 문에 label을 지정하려면 문 앞에 레이블 이름과 콜론이 있어야합니다.

    label:
    statements

break와 continue문은 코드 블록에서 "뛰어 넘을"수 있는 유일한 JavaScript 문입니다.

    통사론

    break labelname;

    continue labelname;

continue문(레이블 참조 유무와 관계없이)은 루프 반복을 건너 뛸때만 사용 할 수 있습니다.

label 참조가 없는 break문은 Loop 또는 switch밖으로 빠져나갈때만 사용할 수 있습니다.

label 참조를 사용하면 break문을사용하여 코드 블록에서 빠져나올 수 있습니다.

    var cars = ["BMW", "Volvo", "Saab", "Ford"];
    list : {
      text += cars[0] + "\n";
      text += cars[1] + "\n";
      break list;
      text += cars[2] + "\n";
      text += cars[3] + "\n";
    }

코드 블록은 {와}사이가 코드 블록입니다.
