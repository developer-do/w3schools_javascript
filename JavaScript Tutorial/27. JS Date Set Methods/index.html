JavaScript Set Date Methods

날짜 설정 메서드를 사용하면 Date object의 날짜 값(년, 월, 일, 시간, 분, 초, 밀리 초)을 설정할 수 있습니다.

날짜 메소드 설정

날짜 설정 메소드는 날짜의 일부를 설정하는 데 사용됩니다.

setFullYear() method

serFullYear() method는 Date 객체의 연도를 설정합니다.

    var d = new Date();
    d.setFullYear(2020);
    document.getElementById("demo").innerHTML = d;

serFullYear() method는 선택적으로 월과 일을 설정할 수 있습니다.

    var d = new Date();
    d.setFullYear(2020, 11, 3);
    document.getElementById("demo1").innerHTML = d;

setMonth() method

setMonth() method는 달(0~11)을 설정합니다.

    var d = new Date();
    d.setMonth(1);
    document.getElementById("demo2").innerHTML = d;

setDate() method

setDate() method는 날짜 개체(1~31)의 일을 설정합니다.

    var d = new Date();
    d.setDate(20);
    document.getElementById("demo3").innerHTML = d;

setDate() method를 사용하여 날짜에 day를 추가할 수 있습니다.

    var d = new Date();
    d.setDate(d.getDate() + 50);
    document.getElementById("demo4").innerHTML = d;

날짜를 추가하거나 월 또는 연도를 이동하면 변경사항은 Date 객체에 의해 자동으로 처리됩니다.

setHours() method

setHours() method는 날짜의 숫자(0~23)의 시간을 설정합니다.

    var d = new Date();
    d.setHours(22);
    document.getElementById("demo5").innerHTML = d;

setMinutes() method

setMinutes() method는 Date의 분을 숫자(0~59)로 설정합니다.

    var d = new Date();
    d.setMinutes(30);
    document.getElementById("demo6").innerHTML = d;

setSeconds() method

setSeconds() method는 Date의 초를 숫자(0~59)로 설정합니다.

    var d = new Date();
    d.setSeconds(30);
    document.getElementById("demo7").innerHTML = d;

날짜 비교

날짜를 쉽게 비교할 수 있습니다.

다음 예제에서는 오늘 날짜를 2100년 1월 14일과 비교합니다.

    var today, someday, text;
    today = new Date();
    someday = new Date();
    someday.setFullYear(2100, 0, 14);

    if(someday > today) {
      text = "Today is before January 14, 2100.";
    } else {
      text = "Today is after January 14, 2100.";
    }
    document.getElementById("demo8").innerHTML = text;

자바스크립트는 0에서 11까지 월을 계산합니다 1월은 0입니다. 12월은 11입니다.
