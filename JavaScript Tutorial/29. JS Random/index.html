JavaScript Random

Math.random()

Math.random()은 0(포함)과 1(포함하지 않음)사이의 난수를 반환합니다.

    Math.random();    // returns a random number

Math.random()은 항상 1보다 낮은 수를 반환합니다.

자바스크립트 무작위 정수

Math.random()은 Math.floor()와 함께 사용되어 임의의 정수를 반환할 수 있습니다.

    Math.floor(Math.random() * 10);   // returns a random integer from 0 to 9
    Math.floor(Math.random() * 11);   // returns a random integer from 0 to 10
    Math.floor(Math.random() * 100);   // returns a random integer from 0 to 99
    Math.floor(Math.random() * 101);   // returns a random integer from 0 to 100
    Math.floor(Math.random() * 10) + 1;   // returns a random integer from 0 to 10
    Math.floor(Math.random() * 100) + 1;   // returns a random integer from 0 to 100

적절한 임의 함수

위의 예제에서 볼 수 있듯이 모든 임의의 정수 용도로 사용할 적절한 임의 함수를 만드는 것이 좋습니다.

이 JavaScript 함수는 항상 min(포함됨)과 max(제외됨) 사이의 난수를 반환합니다.

    function getRndInteger(min, max) {
      return Math.floor(Math.random() * (max - min)) + min;
    }

    function getRndInteger(min, max) {
      return Math.floor(Math.random() * (max - min + 1)) + min;
    }
