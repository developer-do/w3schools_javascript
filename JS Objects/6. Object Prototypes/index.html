JavaScript Object Prototype

모든 JavaScript Object는 prototype에서 property와 method를 상속 받습니다.

이전 장에서는 객체 생성자를 사용하는 방법을 배웠습니다.

    function Person (first, last, age, eyecolor) {
      this.firstName = first;
      this.lastName = last;
      this.age = age;
      this.eyeColor = eyecolor;
    }

    var myFather = new Person("John", "Doe", 50, "blue");
    var myMother = new Person("Sally", "Rally", 48, "green");

또한 기존 객체 생성자에 새 속성을 추가 할 수 없다는 것도 알게 되었습니다.

    추가할 수 없습니다.
    Person.nationality = "English";

생성자에 새 속성을 추가하려면 생성자 함수에 추가해야합니다.

    function Person(first, last, age, eyecolor) {
      this.firstName = first;
      this.lastName = last;
      this.age = age;
      this.eyeColor = eyecolor;
      this.nationality = "English";
      this.fullName = function() {
        return this.firstName + " " + this.lastName;
      }
    }

prototype 상속

모든 JavaScript 객체는 prototype에서 property와 method를 상속받습니다.

Date 객체는 Date.prototype에서 상속받습니다.

Array 객체는 Array.prototype에서 상속받습니다.

Person 객체는 Person.prototype에서 상속받습니다.

Object.prototype은 prototype 상속 chain의 맨 위에 있습니다.

Date Object, Array Object 및 Person Object는 Object.prototype에서 상속받습니다.

Object에 property 및 method 추가

때로는 주어진 유형의 모든 기존 오브젝트에 새 property 또는 method를 추가하려고 합니다.

때로는 객체 생성자에 새 property 또는 method를 추가하려고 합니다.

Prototype property 사용

JavaScript prototype property을 사용하면 객체 생성자에 새 property를 추가 할 수 있습니다.

    function Person(first, last, age, eyecolor) {
      this.firstName = first;
      this.lastName = last;
      this.age = age;
      this.eyeColor = eyecolor;
    }

    Person.prototype.nationality = "English";

JavaScript prototype property를 사용하면 객체 생성자에 새 method를 추가 할 수도 있습니다.

    Person.prototype.name = function() {
      return this.firstName + " " + this.lastName;
    };
