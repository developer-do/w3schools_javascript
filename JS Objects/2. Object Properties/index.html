JavaScript Object property

속성은 모든 JavaScript 객체에서 가장 중요한 부분입니다.

JavaScript property

속성은 JavaScript object와 관련된 값입니다.

JavaScript 객체는 정렬되지 않은 속성의 모음입니다.

속성은 일반적으로 변경, 추가 및 삭제할 수 있지만 일부는 읽기 전용입니다.

JavaScript property 접근하기

객체의 속성에 접근하는 문법은 다음과 같습니다.

    objectName.property       // person.age
    ||
    objectName["property"]    // person["age"]
    ||
    obejctName[expression]    // x = "age"; person[x];

표현식은 특성 이름으로 평가되어야 합니다.

    person.firstName + " is " + person.age + " years old";

    person.["firstName"] + " is " + person.["age"] + " years old";

Loop용 JavaScript

JavaScript for ... in 문은 객체의 속성을 반복합니다.

    통사론
    
    for( variable in object ) {
      // code to be executed
    }

for ... in 루프 내부의 코드 블록은 각 속성에 대해 한 번 실행됩니다.

객체의 속성을 통한 반복:

    var person = {fname: "John", lname: "Doe", age: 25};

    for(x in person) {
      txt += person[x];
    }

새 속성 추가하기

기존 객체에 값을 지정하여 새 객체를 추가 할 수 있습니다.

person 객체가 이미 있다고 가정합니다. 그런다음 새 객체를 제공 할 수 있습니다.

    person.nationality = "English";

속성 또는 메소드 이름에는 예약어를 사용 할 수 없습니다.JavaScript 이름 규칙이 적용됩니다.

속성 삭제하기

delete 키워드는 객체로 부터 속성을 삭제합니다.

    var person = {firstName: "John", lastName: "Doe", age: 50, eyeColor: "blue"};
    delete person.age;

delete 키워드는 속성 값과 속성 자체를 모두 삭제합니다.

삭제 후에 속성을 다시 추가하기 전에 사용할 수 없습니다.

delete 연산자는 객체 속성에서 사용하도록 설계되었습니다. 변수나 함수에는 영향을 미치지 않습니다.

사전 정의된 JavaScript 객체 속성에는 delete 연산자를 사용하면 안됩니다. 그것은 귀하의 응용 프로그램을 손상시킬 수 있습니다.

Property Attributes

모든 property 에는 이름이 있습니다. 또한 그들은 value(값)가 있습니다.

값은 property의 attributes 중 하나입니다.

기타 속성: 열거 가능, 구성 가능 및 쓰기 가능.

이 property는 attributes에 접근 할 수 있는 방법을 정의합니다.(읽기 가능합니까? 쓰기 가능합니까?)

JavaScript에서는 모든 속성을 읽을 수 있지만 value 속성만 변경할 수있습니다.(속성이 쓰기 가능인 경우에만 가능)

(ECMAScript 5에는 모든 property attributes를 가져오고 설정하는 메소드가 있습니다.)

prototype properties

JavaScript 객체는 prototype의 properties를 상속 받습니다.

delete 키워드는 상속 된 속성을 삭제하지 않지만 prototype 속성을 삭제하면 프로토 타입에서 상속 된 모든 객체에 영향을 줍니다.
