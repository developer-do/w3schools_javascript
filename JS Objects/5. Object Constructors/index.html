JavaScript Object Constructors

    function Person(first, last, age, eye){
      this.firstName = first;
      this.lastName = last;
      this.age = age;
      this.eyeColor = eye;
    }

대문자의 첫 문자로 생성자 함수의 이름을 지정하는 것이 좋습니다.

Object Types (Blueprints) (Classes)

위에서 본 예는 한정되어 있다. 그들은 오직 하나의 객체를 만듭니다.

때떄로 우리는 같은 "유형"의 많은 객체를 만들기 위해 "blueprints"가 필요합니다.

"객체 유형"을 만드는 방법은 obejct constructor function를 사용하는 것입니다.

위의 예에서 함수 Person()는 객체 생성자 함수 입니다.

같은 유형의 객체는 new 키워드를 사용하여 생성자 함수를 호출하여 만듭니다.

    var myFather = new Person("John", "Doe", 50, "blue");
    var myMother = new Person("Sally", "Rally", 48, "green");

this keyword

JavaScript에서 this를 호출하는 것은 코드를 "owner"하고 있는 객체입니다.

this의 값은 객체에 사용 될 때, 객체 자체입니다.

constructor function에서 this 값이 없습니다. 그것은 새로운 물건의 컨테이너 입니다. this 값이 새로운 객체가 생성될 때 새로운 객체가 될 것입니다.

참고: this 는 변수가 아닙니다. this는 keyword입니다. this 값을 변경할 수 없습니다.

객체에 속성 추가하기

기존 객체에 새 속성을 추가하는 것은 쉽습니다.

    myFather.nationality = "English";

nationality property가 myFather에 추가됩니다. (myMather에는 추가되지 않습니다.)

생성자에 속성 추가하기

기존 객체에 새 속성을 추가하는 것과 같은 방법으로 객체 생성자에 새 속성을 추가 할 수 없습니다.

    // 안됩니다.
    Person.nationality = "English";

생성자에 새 속성을 추가하려면 생성자 함수에 추가해야 합니다.

    function Person(first, last, age, eyecolor) {
      this.firstName = first;
      this.lastName = last;
      this.age = age;
      this.eyeColor = eyecolor;
      this.nationality = "English";
    }

이 방법으로 객체 속성은 기본값을 가질 수 있습니다.

생성자에 메서드 추가

생성자 함수는 메소드를 정의 할 수도 있습니다.

    function Person(first, last, age, eyecolor){
      this.firstName = first;
      this.lastName = last;
      this.age = age;
      this.eyeColor = eyecolor;
      this.name = function() {return this.firstName + " " + this.lastName;};
    }

기존 객체에 새 메서드를 추가하는 것과 같은 방법으로 객체 생성자에 새 메서드를 추가 할 수 없습니다.

객체에 메소드를 추가하는 작업은 함수 내에서 수행해야 합니다.

    function Person(firstName, lastName, age, eyeColor) {
      this.firstName = firstName;  
      this.lastName = lastName;
      this.age = age;
      this.eyeColor = eyeColor;
      this.changeName = function (name) {
        this.lastName = name;
      };
    }

changeName() 함수는 name 값을 person의 lastName 속성에 할당합니다.

    me.changeName("YoungTae");
