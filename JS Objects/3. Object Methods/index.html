JavaScript Object Methods

    var person = {
      firstName: "John",
      lastName : "Doe",
      id       : 5566,
      fullName : function() {
        return this.firstName + " " + this.lastName;
      }
    };

this keyword

함수 정의에서 this는 함수의 "owner"를 참조합니다.

위의 예에서, this는 fullName 기능을 "소유"하는 사용자 객체입니다.

즉, this.firstName은 이 객체의 firstName property를 의미합니다.

JavaScript methods

JavaScript method는 object에서 수행할 수 있는 액션입니다.

JavaScript method는 function definition를 포함하는 property 입니다.

    var person = {
      firstName : "John",
      lastName  : "Doe",
      age       : 50,
      eyeColor  : "blue",
      fullName  : function() {
        return this.firstName + " " + this.lastName;
      }
    }

method는 object property로 저장되는 function입니다.

Object method access

다음 구문을 사용하여 object method에 접근 합니다.

    obejctName.methodName()

일반적으로 fullName()은 person 객체의 한 method로 fullName은 property로 설명한다.

fullName property는 ()와 함께 호출 될 때 함수로서 실행됩니다.

이 예제는 person object에서 fullName() method에 access합니다.

    name = person.fullName();

()없이 fullName property에 access하면 function definition이 return 됩니다.

    name = person.fullName;

built-in method 사용하기

이 예제에서는 String 객체의 toUpperCase() method를 사용하여 텍스트를 대문자로 변환합니다.

    var message = "Hello world!";
    var x = message.toUpperCase();

adding a method to an Object

객체에 새로운 메소드를 추가 하는 것은 쉽습니다.

    person.name = function() {
      return this.firstName + " " + this.lastName;
    }
