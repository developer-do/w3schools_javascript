요소 스타일 지정

컨테이너의 요소는 style = "position: relative"로 작성해야합니다.

애니메이션의 요소는 style = "position: absolute"로 만들어야합니다.

    #container {
      width: 400px;
      height: 400px;
      position: relative;
      backgrouund: yellow;
    }
    #animate {
      width: 50px;
      height: 50px;
      position: absolute;
      background: red;
    }

애니메이션 코드

기본코드는 다음과 같습니다.

    var id = setInterval(frame, 5);

JavaScript를 사용하여 애니메이션 만들기

    function myMove() {
      var elem = document.getElementById("animate");
      var pos = 0;
      var id = setInterval(frame, 5);
      function frame() {
        if( pos === 350 ) {
          clearInterval(id);
        } else {
          pos++;
          elem.style.top = pos + "px";
          elem.style.left = pos + "px";
        }
      }
    }
