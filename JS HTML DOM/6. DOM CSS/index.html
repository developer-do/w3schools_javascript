JavaScript HTML DOM - Changing CSS

HTML DOM을 사용하면 JavaScript가 HTML 요소의 스타일을 변경 할 수 있습니다.

HTML 스타일 변경하기

HTML요소의 스타일을 변경하려면 다음 구문을 사용하십시오.

		document.getElementById("id").style.property = new style

다음 예제는 p 요소의 스타일을 변경합니다.

		document.getElementById("p2").style.color = "blue";

이벤트 사용

HTML DOM을 사용하면 이벤트가 발생할 때 코드를 실핼할 수 있습니다.

HTML 요소에 "상황이 발생하면" 브라우저에서 이벤트를 생성합니다.

  • 요소에 클릭되었습니다.
  • 페이지가 로드되었습니다.
  • 입력 필드가 변경되었습니다.

이벤트에 대한 자세한 내용은 다음장에서 배웁니다.

이 예제는 사용자가 버튼을 클릭할 대 id = "id1"인 HTML요소의 스타일을 변경합니다.
