JavaScript HTML DOM Elements (Nodes)

node add and remove

새 HTML 요소 (노드) 만들기

HTML DOM에 새 요소를 추가하려면 먼저 요소를 만든다음 기존 요소에 추가해야 합니다.

    

      
This is a paragraph

      
This is another paragraph

예제 설명

    이 코드는 새로운 p 요소를 만듭니다.
    var para = document.createElement("p");

    p요소에 텍스트를 추가하려면 먼저 텍스트 노드를 만들어야 합니다. 이 코드는 테그스 코드를 만듭니다.
    var node = document.createTextNode("This is a new paragraph");

    그런 다음 텍스트 노드를 p 요소에 추가해야합니다.
    para.appendChild(node);

    마지막으로 기존 요소에 새 요소를 추가해야합니다.
    이 코드는 기존 요소를 찾습니다.
    var element = document.getElementById("div1");

    이 코드는 새 요소를 기존 요소에 추가합니다.
    element.appendChild(para);

새 HTML 요소 만들기

앞의 예제에서 appendChild() method는 부모의 마지막 자식으로 요소를 추가했습니다.

insetBefore() method를 사용하지 않으려면 다음을 수행하십시오.

    var para = document.createElement("p");
    var node = document.createTextNode("This is new");
    para.appendChild(node);

    var element = document.getElementById("div1");
    var child = document.getElementById("p1");
    element.insertBefore(para, child);

기존 HTML 요소 제거

HTML 요소를 제거하려면 요소의 부모를 알아야합니다.

    var parent = document.getElementById("div2");
    var child = document.getElementById("p3");
    parent.removeChild(child);

node.remove() method는 IE 브라우저의 모든 버전에서 작동하지 않습니다.

예제 설명

    이 HTML 문서에는 두 개의 자식 노드가 있는 div 요소가 있습니다.
    

      
This is a paragraph

      
This is another paragraph.

    


    id = "div2"인 요소를 찾습니다.
    var parent = document.getElementById("div2");

    id = "p1"인 p 요소를 찾습니다.
    var child = document.getElementById("p1");

    부모로부터 자식을 삭제합니다.
    parent.removeChild(child);

    

      부모를 참조하지 않고 요소를 제거할 수 있으면 좋을 것입니다.

      하지만 DOM은 제거하고자 하는 요소와 그 부모를 모두 알아야합니다.
    

    
일반적인 해결 방법은 다음과 같습니다. 제거할 자식을 찾아 parentNode 속성을 사용하여 부모를 찾습니다.

    
      var child = document.getElementById("p1");
      child.parentNode.removeChild(child);

HTML 요소 바꾸기

요소를 HTML DOM으로 바꾸려면 replaceChild() method를 사용합니다.

    var para = document.createElement("p");
    var node = document.createTextNode("This is new.");
    para.appendChild(node);

    var parent = document.getElementById("div1");
    var child = document.getElementById("p1");
    parent.replaceChild(para, child);
