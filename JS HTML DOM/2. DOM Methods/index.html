JavaScript - HTML DOM Methods

DOM 프로그래밍 인터페이스

HTML DOM은 JAvaScript(및 다른 프로그래밍 언어)로 접근 할 수 있습니다.

DOM에서 모든 HTML 요소는 Object로 정의됩니다.

프로그래밍 인터페이스는 각 객체의 속성 및 메서드입니다.

property는 (HTML 요소의 내용을 변경하는 등) 가져 오거나 설정할 수 있는 값입니다.

method는 (추가하거나 HTML요소를 삭제 같은) 내가 할 수 있는 작업이다.

Example

다음 예제에서는 id = "demo"인 < p > 요소의 내용 (innerHTML)을 변경합니다. 

    p id="demo1"

위의 예에서 getElementById는 method 이고 innerHTML은 property 입니다.

getElementById method

HTML 요소에 액세스하는 가장 일반적인 방법은 요소의 ID를 사용하는 것 입니다.

위의 예제에서 getElementById method는 요소를 찾기 위해 id = "demo"를 사용했습니다.

innerHTML proeprty

요소의 내용을 가져오는 가장 쉬운 방법은 innerHTML속성을 사용하는 것 입니다.

innerHTML 속성은 HTML요소의 내용을 가져오거나 바꾸는 데 유용합니다.

innerHTML 속성은 html 및 body를 포함한 모든 HTML요소를 가져오거나 변경하는 데 사용할 수 있습니다.
