JavaScript HTML DOM - Changing HTML

HTML DOM을 사용하면 JavaScript가 HTML 요소의 내용을 변경할 수 있습니다.

HTML 출력 스트림 변경하기

JavaScript는 동적 HTML 콘텐츠를 생성할 수 있습니다.

JavaScript에서는 document.write()를 사용하여 HTML 출력 스트림에 직접 쓸쑤 있습니다.

    document.getElementById("dateTime").innerHTML = Date();

HTML 내용 변경하기

HTML 요소의 내용을 수정하는 가장 쉬운 방법은 innerHTML 속성을 사용하는 것 입니다.

HTML 요소의 내용을 변경하려면 다음 구문을 사용하십시오

    document.getElementById(id).innerHTML = new HTML;

이 예제는 p 요소의 내용을 변경합니다.

    document.getElementById("p1").innerHTML = "New text!!";

예제 설명

  • 위의 HTML 문서는 id가 "p1"인 p 요소를 포함합니다.
  • HTML DOM을 사용하여 id="p1"이 요소를 가져옵니다.
  • JavaScript는 해당 요소의 내용(innerHTML)을 "New text!!"로 변경합니다.

이 예제는 h1 요소의 내용을 변경합니다.

Old Heading

    < h1 id="id01">Old Heading< /h1 >

    var element = document.getElementById("id01");
    element.innerHTML = "New Heading";

예제 설명

  • 위의 HTML 문서에는 id가 "id01"인 h1 요소가포함되어 있습니다.
  • HTML DOM을 사용하여 id = "id01"인 요소를 가져옵니다.
  • JavaScript는 해당 요소의 내용(innerHTML)을 "New Heading"

속성값 변경

HTML 속성 값을 변경하려면 다음 구문을 사용하십시오.

    document.getElementById(id).attribute = new value;

이 예제는 img의 요소의 src특성 값을 변경합니다.

    function changeFunc() {
      document.getElementById("myImage").src = "landscape.jpg";
    }
