JavaScript HTML DOM Navigation

HTML DOM을 사용하면 노드 관계를 사용하여 노트 트리를 탐색할 수 있습니다.

W3C HTML DOM 표준에 따르면 HTML 문서의 모든 것이 노드입니다.

  • 전체 문서가 문서 노드입니다.
  • 모든 HTML 요소는 요소 노드입니다.
  • HTML 요소 내부의 텍스트는 텍스트 노드입니다.
  • 모든 HTML 속성은 속성 노드 입니다.(사용되지 않음)
  • 모든 주석은 주석 노드 입니다.

HTML DOM을 사용하면 노드트리의 모든 노드에 JavaScript로 접근할 수 있습니다.

새 노드를 만들 수 있으며 모든 노드를 수정하거나 삭제할 수 있습니다.

노드 관계

노드 트리의 노드는 서로 계층 관계를 갖습니다.

상위, 하위 및 형제라는 용어는 관계를 설명하는데 사용됩니다.

  • 노드 트리에서 최상위 노드는 root라고 부른다 (또는 node root)
  • 모든노드에는 root를 제외한 정확히 하나의 상위 노드가 있다(상위 없음).
  • 노드는 많은 아이들을 가질 수 있습니다.
  • 형제는 동일한 부모를 가진 노드입니다.

위의 HTML에서 다음을 읽을 수 있습니다.

  • html은 root node입니다.
  • html에는 부모가 없습니다.
  • html은 head및 body의 부모입니다.
  • head는 html의 첫번째 하위 항목입니다.
  • body는 html의 마지막 자식입니다.

그리고

  • head에는 하나의 title 자식이 있습니다
  • title에는 하나의 자식 text node가 있씁니다. "DOM Tutorial"
  • body에는 두 개의 하위 항목이 있습니다. h1 및 p
  • h1에는 "DOM Lesson one"이라는 하나의 text node인 자식이 있습니다.
  • p에는 "Hello World!"이라는 하나의 text node인 자식가 있습니다.
  • h1 및 p는 형제입니다.

노드 간 이동

다음 노드 특성을 사용하여 JavaScript로 노드 사이를 탐색할 수 있습니다.

  • parentNode
  • childNodes[node number]
  • firstChild
  • lastChild
  • nextSibling
  • prevSibling

하위 노드 및 노드 값

DOM 처리의 일반적인 오류는 요소 노드에 텍스트가 포함될 것으로 예상되는 것입니다.

< title id="demo" >DOM Tutorial < /title >

요소 노드 title에는 텍스트가 들어 있지 않습니다.

여기에는 "DOM Tutorial"값이 있는 text node가 있습니다.

텍스트 노드의 값은 노드의 innerHTML 속성을 통해 액세스 할 수 있습니다.

    var myTitle = document.getElementById("demo").innerHTML;

innerHTML 속성은 접근하는 것은 첫 번째 자식의 nodeValue에 접근하는 것과 같습니다.

    var myTitle = document.getElementById("demo").firstChild.nodeValue;

첫 번쨰 자식에 대한 접근은 다음과 같습니다.

    var myTitle = document.getElementById("demo").childNodes[0].nodeValue;

InnerHTML

이 자습서에는 innerHTML 속성을 사용하여 HTML요소의 내용을 검색합니다.

그러나 위의 다른 방법을 배우는 것은 트리구조와 DOM 탐색을 이해하는 데 유용합니다.

DOM root node

전체 문서에 대한 접근을 허용하는 두가지 특수 속성이 있습니다.

  • document.body - 문서의 본문
  • document.documentElement - 전체 문서
    console.log(document.body.innerHTML);
    console.log(document.documentElement.innerHTML);

nodeName 속성

nodeName 특성은 노드의 이름을 지정합니다.

  • nodeName은 읽기 전용입니다.
  • 요소 노드의 nodeName은 태그 이름과 같습니다.
  • 속성 노드의 nodeName은 속성 이름입니다.
  • 텍스트 노드의 nodeName은 항상 #text입니다.
  • 문서 노드의 nodeName은 항상 #document입니다.

My First Page

    document.getElementById("id03").innerHTML = document.getElementById("id04").nodeName;

tip: nodeName은 항상 HTML 요소의 대문자 태그 이름을 포함합니다.

nodeValue 등록정보

nodeValue 등록 정보는 노드 값을 지정합니다.

  • element node에 대한 nodeValue가 null입니다.
  • text node의 nodeValue가 text 그 자체입니다.
  • attribute에 대한 nodeValue가 attribute 값 입니다.

nodeType 특성

nodeType 등록정보는 읽기 전용입니다. 노드의 유형을 리턴합니다.

    document.getElementById("id02").innerHTML = document.getElementById("id01").nodeType;

nodeType등록 정보를 보시고싶으시면 여기를 클릭하시고 가장 아래쪽 문서를 확인해주세요.
