JavaScript HTML DOM Elements

이 페이지는 HTML 페이지에서 HTML element를 찾아 접근하는 방법을 알려줍니다.

HTML 요소 찾기

흔희 JavaScript를 사용하면 HTML 요소를 조작하려고 합니다.

이렇게 하려면 요소를 먼저 찾아야 합니다.

  • ID로 HTML 요소 찾기
  • 태그 이름으로 HTML 요소 찾기
  • 클래스 이름으로 HTML요소 찾기
  • CSS 선택자로 HTML 요소 찾기
  • HTML 객체 컬렉션을 사용하여 HTML 요소 찾기

ID로 HTML 요소 찾기

DOM에서 HTML 요소를 찾는 가장 쉬운 방법은 요소 id를 사용하는 것입니다.

이 예제에서는 id = "intro"인 요소를 찾습니다.

    var myElement = document.getElementById("intro");

요소가 발견되면 method는 요소를 myElement에서 반환합니다.

요소가 발견되지 않는 경우, myElement는 null을 포함합니다.

태그 이름으로 HTML 요소 찾기

이 예제는 모든 < p > 요소를 찾습니다.

    var x = document.getElementsByTagName("p");

이 예제는 id = "main"인 요소를 찾은 다음에 "main"내부의 모든 p 요소를 찾습니다.

    var x = document.getElementById("main");
    var y = x.getElementsByTagName("p");

클래스 이름으로 HTML 요소 찾기

동일한 클래스 이름을 가진 모든 HTML 요소를 찾으려면 getELementsByClassName() 을 사용하십시오.

이 예에서는 class = "intro"인 모든 요소의 목록을 반환합니다.

    var x = document.getElementsByClassName("intro");

IE 8 및 이전 버전에서는 클래스 이름별로 요소를 찾지 못합니다.

CSS 선택자로 HTML 요소 찾기

지정된 CSS 선택기 (ID, 클래스 이름, 유형 속성, 속성 값 등)와 일치하는 모든 HTML 요소를 찾으려면 querySelectorAll() method를 사용하십시오.

이 예제는 class="intro"인 모든 p 요소의 목록을 반환합니다.

    var x = document.querySelectorAll("p.intro");

IE 8 및 이전버전에서는 querySelectorAll() method가 작동하지 않습니다.

HTML 객체 컬렉션을 사용하여 HTML 요소 찾기

이 예에서는 forms 컬렉션에서 id = "frm1"인 양식 요소를 찾고 모든 요소값을 표시합니다.

    var x = document.forms["frm1"];
    var text = "";
    var i;
    for(i = 0; i < x.length; i++) {
      text += x.elements[i].value + "
";
    }
